I've completed much improved version that is now part of Singularity
Alpha builds that can be downloaded from

https://files.streamgrid.net/singularity/

It supports texture export, applying texture parameters (repets, flip,
offset) to exported UV maps, merging faces with same texture, slipping
transparent faces, etc-
