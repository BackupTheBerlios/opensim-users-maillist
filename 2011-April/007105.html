Hi missy,

I added a note about this to the Configuration page on the Opensim wiki, my
guess is it's the MySQL version you're running.  Here's the note from the
wiki:

Some users have reported problems with MySQL 5.1.55 and up with Opensim
0.7.0.2, see this
thread<http://opensim-users.2152040.n2.nabble.com/OpenSim-1-7-0-2-and-MySQL-Versions-td6155064.html>
for
more information. The issue can be resolved by using an updated version of
MySql.Data.dll OR installing an older version of MySQL such as MySQL
5.1.52<http://downloads.mysql.com/archives.php?p=mysql-5.1&v=5.1.52>
with
Opensim version 0.7.0.2.

Best of luck!
