If you're able to build and test with the very latest OpenSim git master, then I inserted some log lines a week ago that 
tell you which config files it's loading.  Commit was d8e1c38.  Look out for

[LOAD REGIONS PLUGIN]: Loading region configurations from filesystem

and the lines that follow.  If there are no following lines then the config files aren't being found at all.

The latest osgrid distro should also have these changes.
