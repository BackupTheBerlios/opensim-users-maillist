Hi Cider,

What database are you using?  If MySQL, I think it's well worth installing
phpMyAdmin http://www.phpmyadmin.net/home_page/index.php as that gives you a
nice GUI for making database edits.  It's far more user friendly than doing
command line stuff when working with MySQL, and chances are this won't be
the only time you want to edit the database directly.  You can make direct
edits via the command line too, you just have to be super careful with the
syntax so you don't hose things and if you don't feel comfortable generally,
I'd suggest the GUI is the better way to go.
