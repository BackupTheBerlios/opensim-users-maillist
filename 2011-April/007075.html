Sometimes you run into an issue with the 'Useragent' that is being
used for HTTP-requests from a script to an (external) http-server.
Some http-servers require a certain keyword (like 'Mozilla') in the
user-agent header. There is a setting in the OpenSim configuration to
change this useragent field.

Not sure if this is a solution to your problem, but hope it gets you somewhere!
