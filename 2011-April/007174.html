No guarantee, but there's a reasonably old hack that might apply to Imprudence:
http://forums.osgrid.org/viewtopic.php?f=4&t=175
The link shows how to set the limit in the old SL viewer to 100m, but
of course you can set it to any size you want.

I had tried it with the old SL viewer a couple of years ago on an OS
grid, I can only assume Imprudence has some similar file you could
alter in the same way. So far 256^3 has been plenty big for me, but
with the advent of larger sims a-la-Aurora-Sim that may change...

Have fun,
