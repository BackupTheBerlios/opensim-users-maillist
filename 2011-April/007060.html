well CenomeCache is not very stable within opensim.  
And if you use more assets then configured in CenomeCache, opensim will
crash and assets are gone..
with flotsam-cache, its saved on your HD. In case of crash and you restart
again, you can retrieve some
of your assets (depend if everything was cached before the crash). 

One good tip to prevent data loss is to type the "backup" command in
console, this will save everything to the database. 
There's a timer option within OpenSimDefault.ini 
    ; To run a script every few minutes, set the script filename here
    ; timer_Script = "filename"

so if you have regions with high prim count, use flotsam cache..
