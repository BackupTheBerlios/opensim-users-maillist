Right now, the only way you can get a certain amount of security is 
running a closed-garden grid with the services inside a firewall.
Some of us are looking into getting that same amount of security in the 
Hypergrid and other open configurations, that is, with the services on 
the public internet and people being able to visit arbitrary grids. But 
that's still not here. So if you want security right now, you need a 
firewall.
