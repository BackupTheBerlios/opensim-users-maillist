Hi,

you can do that using realXtend enhancements: modrex to make the opensim
support them, and realXtend viewer to view swf and vnc on a prim. You can
use realxtend to render swf to hud in interactive mode, though...
If you don't get modrex to work (it is a bit alphaish atm), you can use
realXtend standalone server: Download link:
http://www.cybertechnews.org/?page_id=384

swf support on a prim is a bit limited, you can not interact with it, but
still there are some good usecases, like:
show youtube videos: http://www.cybertechnews.org/?p=369
show powerpoints: http://www.cybertechnews.org/?p=599

VNC has been working, but I need to test it myself first to see if it works
with the latest 0.4 release.
