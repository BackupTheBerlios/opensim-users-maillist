Hi Kevin

I'm running both 0.6.6 and Diva's megaregion release successfully
under Linux at work.

However, I've had no end of problems with the region handshake. If you
want to, you can read all about my experiences at
http://www.linuxtraining.org.uk/blogger4.html

Obviously my network connectivity is very different from yours but
I've come to the conclusion that if you want OpenSim to be accessible
externally, then the ExternalHostName parameter in Regions.ini must be
the external IP address (accessible over the 'Net).

Hope you finally get it working :)
