The number of regions on opensim can host a the same time is dependant on
the hardware prims, scripts and the number of avatars online.

In standalone the max seems te be around 20 void regions with 20 avatars
online on the sims.
In grid mode with the server running on different machines you can run more
than a lot of full regions with 80 avatars online, as long as you have got
enough machines to support the regions

The locations are offscale, you should not place the regions to far apart.
