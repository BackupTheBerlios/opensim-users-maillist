the only issue i see with your region set up if you are trying to access
from outside your lan is the InternalAddress ( either 0.0.0.0 or the
internal IP of your server e.g. 192.168.1.2 ) and the ExternalHostName is
set to the external Static IP of your system ( visit
http://whatismyip.comto get ip information for your system. as far as
how many regions you can
run in standalone mode i have done up to 400 on 512 mb of ram ( not
adviseable) but in grid mode you should allow at least 1 GB of ram per
region has worked best for me hope i have not confused you or just readded
previous information to this post.
