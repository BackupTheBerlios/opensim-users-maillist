to fully support HTTP Inventory.

JCC mentioned the opensim version differences in a kokua dev meetup several weeks ago.

I don;t know opensim well enough to comment on which versions should not use HTTP Inventory and which should.

If the caps are not present the viewer and should fallback to UDP.
