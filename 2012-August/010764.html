User CALs will not effect OpenSimulator, its more for things like Remote
Access and accessing other Microsoft services on the machine such as Print
Sharing and File Storage, you can read more about it here :
https://www.microsoft.com/licensing/about-licensing/client-access-license.aspx#tab_2the
only thing you really need to be concerned with is selecting a version
that supports your hardware properly.

http://msdn.microsoft.com/en-us/library/windows/desktop/aa366778%28v=vs.85%29.aspx#physical_memory_limits_windows_server_2008_r2

you will need Windows Server 2008 R2 Enterprise or Datacenter
