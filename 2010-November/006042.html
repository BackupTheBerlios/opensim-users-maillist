To make your avatar not be a cloud, you must create a new set of avatar body
parts using the client in your inventory and then wear them.

If you're using LibOMV, it's more complicated.  You'll need to either create
the shape and outfit manually with libOMV or log in with the client, create
them, wear them.  After that, log out with the client and log-in with libOMV
and use the Set Last Appearance function.

-Teravus
