Hi Andy,

I think this page on the wiki should have all of the information you need:

http://opensimulator.org/wiki/Connecting

Basically you need to choose a viewer and then depending upon whether or not
that viewer has a Grid Manager, you'll enter the information for your
opensim install and then you should be able to connect.

If you aren't running Diva Distro, make sure that you've created a user from
the console so you have an account to log into!

Good luck!
