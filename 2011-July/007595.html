On 30/06/11 20:21, Fan Jun Wei wrote:
> Hello,
> I would like to configure the user levels for selected users and raise it up to 200 to allow them to use god mode. Also,
> how do I restrict permissions of standard users, for example, disabling the creation of content in selected areas and
> disabling the ability to fly? Please help.
> Thank you,
> fanjw

In the latest development OpenSim code, there is a command called set user level on the console which will allow you to 
change this.

In 0.7.1.1 you will need to change it manually in the database.  There is some information on this at

http://opensim-users.2152040.n2.nabble.com/how-to-give-user-full-admin-god-powers-td6294154.html

Disabling content creation and flying can be done via the viewer in-world, as long as that user is the estate owner or 
manager, I believe.
