On 14/07/11 15:54, Kay McLennan wrote:
> I am interested in an update on the integration of mesh in the OpenSim (?).  More specifically, does the current
> release of OpenSim (0.7.1.1) enhance the use of mesh in OS virtual worlds?  Also, any recommendations for exemplary mesh
> builds [in a public OS grid] to view?

0.7.1.1 has an implementation of mesh but it's the old Linden Lab beta mechanism.  You'll only be able to upload mesh 
with older SL beta viewers though I believe all viewers can still currently see the older mesh.

There is an implementation of what is currently the newer mesh mechanism but it's only in OpenSim development code.  If 
you can, I would suggest waiting until 0.7.2 is released.  I hope this can wait until the mesh implementation is 
finalized (i.e. rolled out on the main Linden grid in a couple of months) but if that's delayed then we can just release 
anyway.
