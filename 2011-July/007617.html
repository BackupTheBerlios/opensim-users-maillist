Mumbles install package is confusion because if you install it you actually
get a server (murmur.exe) and a client (mumbles.exe) all in a folder called
mumbles. I just installed the mumbles install package (also called
mumbles.exe) on my opensim server box and went and started up the murmur.exe
from that folder. When you do that you now have a voice "server" running.
After that is running you can start up opens and it will find that and hook
into it ( you can see that in the console).
