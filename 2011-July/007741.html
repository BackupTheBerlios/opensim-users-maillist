Hi Per :)

It should be possible to do sales for L$0.  It's necessary to point the -helperuri switch in the viewer to the same 
address as your standalone.

I'm not totally sure how this is done for a grid system - whether an external module is needed or not.  This area is 
extremely (i.e. not) documented.  You can see how the -helperuri is set on the osgrid login page

http://www.osgrid.org/index.php/viewerconnect

The underlying backend might be landtool in this instance.

Setting grid gods should work, presuming that you also make the user a god.  On 0.7.1.1 this has to be done by setting 
the UserLevel field in the UserAccounts table to 200 or above.  On current git master there is a region console command 
to do this.

As for the RG groups module, I don't believe this was ever actually open-sourced in the end.
