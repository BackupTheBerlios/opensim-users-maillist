the biggest problem is there are basically 3 things that need backing up

1 the opensim program files
2 your database(s)
3 the actual assets

1 is simple just keep a copy of the original download (with any
settings changes)
2 is a bit fun but there are tools that can be used (and automated)
3 is the real killer this can be done by A a file tree backup B oar
files of each region and IAR files of each accounts inventory
