At work we're running in grid mode on a Dell PowerEdge R710 and Windows
Server 2008 R2 Standard (64-bit) with no problems.  As the other Chris
noted, just make sure to run OpenSim.32BitLaunch.exe instead of Opensim.exe.

I don't have an NVidia video card, so can't help with that one.  Good luck
with your upgrade though!  :)
