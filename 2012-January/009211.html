This facility was very probably non-functional in OpenSimulator 0.7.2 and quite a few releases before.

In current master code it is working again.  I have updated the instructions to match (I had no idea this page existed).

However, this is a stress-testing facility.  It is almost certainly not what you're looking for to create bots, though 
it may be useful as example code.

But to be blunt, if you can't get some idea of how thinks working from reading the wiki then this is not for you. 
Setting up bots is a complicated and technical process, and facilities like OSSLNPC should be considered experimental.
