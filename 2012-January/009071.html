I use Dolphin 3 and have found Lance Corrimal to be receptive to edu- and OpenSim-related suggestions. He is currently looking at the chatted hypergrid address issue which none of the version 2/3 viewers support afaik (I tried Exodus, Niran's and Firestorm). Dolphin also has a useful grid dropdown if not yet a grid manager (you have to hack a small XML file to add a grid although he will most likely add any signiicant ones).



It handles MOAP fine and displays but does not upload mesh. So I use the SL Viewer 3 for uploading and the students use Dolphin.



The only issue I have had with it is that it tends not to play nicely with avatars with large inventories.



Best wishes



Peter
