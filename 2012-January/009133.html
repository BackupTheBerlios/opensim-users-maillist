I personally have not experienced any (major) issues running OpenSim 
server and / or SecondLife viewers  on Win 7 x64. Just make sure to run 
OpenSim.32BitLaunch.exe instead of the standard OpenSim.exe to run the 
OS server. I'm not sure about grid mode as I have been using standalone 
for some time now.

As far as video performance, I personally use the GTX 560 Ti and it runs 
on Ultra settings beautifully. It will also run shadows and deferred 
rendering albeit at a slightly low FPS (I usually top around 15 - 20 FPS 
with shadows on). So my guess would be the GTX 580 would be a perfectly 
reasonable choice.
