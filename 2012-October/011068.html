[AGENT_NAME] is a tag that the V3 uses in it's macro language. The V3 
profile is a web app that will take the user name firstname.lastname as 
an argument. The viewer must have the patches to support it as well. 
This is a work in progress and needs to have the OAuth parts added so 
the viewer can get a token in the login response that it can use to 
login to the web app for the account owner to do edits.
