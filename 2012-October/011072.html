Our PMgrid didn't display map tiles using Firestorm (V3 based) so your link
was useful. After we implemented the changes in the link I logged in to
PMgrid and hey presto the map displayed [btw: this displays faster than with
the Imprudence (v1 based) viewer].

I did a little more testing on this and wanted to share my findings. 

1) FIRESTORM: Log into PMgrid and the PMgrid grid map displays fine. My
PMgrid avatar then HGs to UOBgrid and it does not display map tiles in
UOBgrid.

2) FIRESTORM: Log into UOBgrid and the UOB grid map displays fine (so they
must have those v3 map changes in). My UOB avatar then HGs to PMgrid and it
does not display map tiles in PMgrid.

3) IMPRUDENCE: Log into PMgrid and the PMgrid grid map displays fine. My
PMgrid avatar then HGs to UOBgrid and it displays map tiles in UOBgrid no
problem.

4) IMPRUDENCE: Log into UOBgrid and the UOBgrid grid map displays fine. My
UOBgrid avatar then HGs to PMgrid and it displays map tiles in PMgrid no
problem.

It seems that whilst the v1 map tiling works via hypergrid the v2/v3 one
doesn't. Can this be fixed easily?
