*SCOPE OF WORK*

Designing Digitally Inc. has purchased a dedicated server and is currently
running 0.5.9 OpenSim via standalone. Currently the system is able to handle
what DDINC is currently doing; however there are additional requirements for
this private grid OpenSim that DDINC would like to see implemented. These
items are needed to improve the current system in hope to provide a
professional private meeting space for DDINC.


The contractor will log into the Windows dedicated server using Windows
Remote Desktop. The contractor will install and setup the newest stable
version of OpenSim 0.6.3+ as a grid system that can be easily scaled up.

The contractor will compile the new version of OpenSim with the VOIP
Integration (FreeSwitch).

The contractor will setup the Web Statistics Module which will allow DDINC
to view those stats on the HTML/PHP web server on the Windows Dedicated
Server.

The contractor will setup a database (databases provided by DDINC)  for each
of the following services: User, Grid, Asset, Inventory, and Messaging or
UGAIM .

The contractor will setup the HTML/PHP page for DDINC to be able to register
(register page) users to their grid.

The contractor will setup the HTML/PHP page for users to preview (login
page) when attempting to log into DDINC grid.

The contractor will setup the HTML/PHP page for users to obtain their lost
password.

The contractor will test the OpenSim to ensure the scripting, physics,
groups, registration, VIOP, Web Stats are up and running correctly without
any disruptions.



*SPECIFICATIONS*

The Contractor will provide a per project price for the entire scope of work
provided above.

The Contractor will provide a timeline for this development

The Contractor will provide examples of previous work regarding setup of the
OpenSim.
