 It has been a while since I've messed around with it, but unless OpenSim is
using a patched version of Nini, people need to be *very* careful with these
console commands, particularly "config save". Nini is marvelous at reading
INI files, but has some pretty nasty issues writing to them. If I remember
right, it tends to strip quotes, for example, potentially destroying stuff
like DB connection strings.
