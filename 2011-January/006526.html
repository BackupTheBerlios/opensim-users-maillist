On startup of OpenSimulator, Wifi will create (or update, if necessary) the administrator account specified by the configuration options AdminFirst, AdminLast, AdminPassword, and AdminEmail in section [WifiService]. (For the Diva distribution, these options are set in bin/config-include/MyWorld.ini and can be changed by running Configure.exe.) The latter two of these options are not mandatory; if no AdminPassword is specified, Wifi will ask for one when you access the web interface.

If you already have an administrator account and want to give Wifi admin privileges to another account, then you can do this in the web interface by changing the "Level" of that other account to a value of 100 or greater.
