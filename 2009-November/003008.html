Hi Andrew,

this is because there is no config file by default. You need to make a copy
of your OpenSim.ini.example file (which is in the same folder as
OpenSim.exe), rename it for OpenSim.ini and edit it to fit your network and
database settings.
Then it should start fine ;)

Olish.
