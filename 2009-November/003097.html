FWIW I now have HG working OK.  I use ports 9000-93xx for the various grids
and regions, and for the Opensim.ini for each grid, I've replaced all of the
grid port references that are set to 127.0.0.1 by default with mydomain.com,
and I've changed all the 800x ports to the HTTP listener port (9110 and 9300
for the 2 separate grids I'm using to test in my case).  On the router I've
just closed all the 800x ports and it all still works fine.  No other ports
are open apart from 90xx to 93xx and the 'default' ones (port 80 etc.).

One thing - yesterday, while I was trying to solve my incompatible
coordinates problem, I made some changes to opensim.ini which caused the
login to hang at "Waiting for region handshake". I reverted Opensim.ini to
its previous configuration but the region handshake problem persisted.  To
clear it, I had to also revert the Opensim MySQL database to its previous
instance, after which it all worked perfectly again. - Regards, Kevin
