I would suggest trying the latest Diva release at least as a starting point. Diva has set up a very good configuration script in which just about everything works. Maybe not exactly the way you want, but it WILL work!

The general releases are more flexible, but can be very difficult to configure so as to avoid strange things happening (or not happening). The documentation is non too clear until you have actually run things, and worse yet there are things that work differently now!

Don't forget to back up your database BTW - I didn't and I am still restoring things from source files after two months!
