You want to search for tools which allow real time presentation over the web, since it's the web you are using from 
media on a prim and not the simulator itself.

A quick google for "web real time presentation" brings up a few, such as blacktonic and prezi.  I've no idea how good 
any of these tools are.

Alternatively, for presentations in particular I know there are various LSL simulator tools (not media on a prim) which 
implement a slideshow.  Unfortunately, I can't help with specifics of where to find these for OpenSim or whether any are 
available in IARs.  On a quick google, there's a very primitive script at [1] but I didn't write it so I can't help 
further (and part is in german!).

I'm not terribly surprised youtube doesn't appear simultaneously.  Different computers will take different speeds to 
load and play the video.

[1] http://opensimulator.org/wiki/Presentation_Board
