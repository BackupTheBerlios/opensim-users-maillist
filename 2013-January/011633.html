Yeah --home was --profile up until 0.7.3.

I must admit, I haven't look at --home for a while but from what R.Gunther is saying it sounds like it works.  I believe 
it places the url (in this case http://www.osgrid.org) in the creator information - I don't know if it then appends 
anything else.

 From looking at [1], the original intention may have been to make this a link to each user's profile on the 'home' grid 
given.  However, I'm not sure if this work has reached a point where one could just use that URL to see the original 
user's profile, esp. as profile support seems rather sketchy atm.

[1] http://opensimulator.org/wiki/OpenSim_Archives#--home
