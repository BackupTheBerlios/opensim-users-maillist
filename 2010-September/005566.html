This sounds like the texture file has become disconnected from the avatar
somehow, possibly it never got into the actual database for some reason.

Due to cacheing in the viewer this may not be visible until you switch
viewers or reload the cache.

This is a common problem when transferring items between grids via an XML
file, unless the textures are saved and uploaded with the XML file
everythign looks great until you look at it with another viewer.
