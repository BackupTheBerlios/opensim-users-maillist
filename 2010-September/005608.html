It appears that many Linux distros now ship with only the "mono-runtime"
packages installed, but many OpenSimulator features such as LSL scripts
require "mono-complete" installed.

"mono-complete" Includes the development packages which are needed to
compile and run scripts.

It is suggested that all Linux users install the "mono-complete" package
with at least mono 2.4.2 (Ubuntu 10.04 ships with mono 2.4.4 which runs
fine).
