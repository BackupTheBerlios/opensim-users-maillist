DutchGlory wrote:
>
> so you need to setup opensim.ini AND OpenSimDefaults.ini  doesn't sound
> logic.  there has to be a more efficient way to load configuration.

You are not supposed to change OpenSimDefaults.ini at all. As the name says, it sets the defaults which then can be overridden by settings in OpenSim.ini.

If you want to change any setting in OpenSimDefaults.ini that you can't find in OpenSim.ini, then you *copy* the appropriate parameter from OpenSimDefaults.ini to OpenSim.ini and adjust it there.
