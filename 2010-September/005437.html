On 01/09/10 00:21, Kesick, Nicholas Charles wrote:
> I noticed when starting OpenSim that it says “Console Logging Level is
> All”.
>
> How can you change the console logging level?

Either change it in OpenSim.exe.config (it's a log4net config) or do

# set log level <level>

on the console (see opensim help on the console for more details).  However, changes made on the console will be lost on 
restart.
