Hi CJ,

For question #3, not sure if you're running your grid HG enabled or not,
but for me, the settings to choose the default landing region (and a
fallback region if you choose) is in Robust.HG.ini settings in
the [GridService] section:

;; Next, we can specify properties of regions, including default and
fallback regions
    ;; The syntax is: Region_<RegionName> = "<flags>"
    ;; or:            Region_<RegionID> = "<flags>"
    ;; where <flags> can be DefaultRegion, FallbackRegion, NoDirectLogin,
Persistent, LockedOut,Reservation,NoMove,Authenticate
    ;; For example:
    Region_FleepGrid_Plaza = "DefaultRegion"
    Region_Chilbo = "FallbackRegion"
    ; (replace spaces with underscore)

As to question #2, AFAIK the only way to indicate whether teleports are
allowed anywhere in the region or are blocked is through the viewer in the
Estate settings.

Hope that helps!
