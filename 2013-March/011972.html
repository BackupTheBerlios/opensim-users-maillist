This code was donated a while ago but I'm not sure if it works, and no fixes
have been submitted.

To be honest, though I originally put it in, I'm not sure now whether it's
appropriate for core, as any backups taken by this method except at
extremely long intervals are quickly going to be enormous (unless one keeps
deleting the old ones, which would kind of negate the whole point).

I would be interested to know people's thoughts on this.
