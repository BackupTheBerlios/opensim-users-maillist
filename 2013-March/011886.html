In principle it might be possible to get the MOAP module to not serve shared media surfaces to users in other plots but 
this wouldn't work if the avatar was crossing plot lines (since it would already have the data).

I think one would really need the viewer to actively drop MOAP surfaces in other plots (in which case you wouldn't need 
any simulator-side changes).

I would actually be kind of surprised if there isn't a TPV that offers something like this, since MOAP is heavyweight 
and afaik the display limit is 5 surfaces.  But I haven't explored that.

This question actually finally triggered me to create a crude diagram of how MOAP works [1], namely that the 
communication is directly with a website.  But I don't think that actually has anything to do with this question.

[1] http://opensimulator.org/wiki/Media_On_A_Prim
