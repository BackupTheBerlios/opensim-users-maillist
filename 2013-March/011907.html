That's a handy command to know Justin, thanks - it showed it was still
set to true, which I traced to DivaPreferences.ini, so I copied it
into MyWorld.ini and set it to false. It appears to have worked but I
can't tell for sure now because I'm still dealing with another issue
(logging into or TPing to multi-regions (formerly(?) megaregions), I
land in the 'seams' between sims and fall into the void).

It looks like it would be a good idea, Diva, to make only a single
file to edit, but as OpenSim.ini is just a commented-out duplicate of
OpenSimDefaults.ini, I've been using that as well because it has many
settings not included in the MyWorld.ini. Of course MyWorld.ini
doesn't include the CombineContiguousRegions command either (which
DivaPreferences.ini does), so I made the (mistaken) assumption that
megaregions had already been set/unset in OpenSim.ini; I hadn't even
looked at DivaPreferences, again (mistakenly) assuming MyWorld was
just a duplicate of that. Live and learn!

So I now have CombineContiguousRegions set in 4 config files!
