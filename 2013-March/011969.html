Hi Rina,

I've played around a little bit with this myself.  With my situation, I
was looking at more 'nature' type noises,for which there are two tricks
that I use.

For certain noises such as wind that need to play constantly, I just get
the sound that I want to use, and then use llLoopSound to get it going.

For sounds that need to play a little less constantly, like birds
chirping, I use a timer script and then place objects at different places
around the region.  I tend to change the timer length somewhat so that
they're not all using the same interval (so some might trigger at 60
seconds, some at 90 seconds for instance) just to vary things up a little
bit.

Hope this helps.
