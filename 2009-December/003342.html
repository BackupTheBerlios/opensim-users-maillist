If you are running it all local and trying to connect from another local
machine then edit the region info to 0.0.0.0 internal and whatever the ip
address is for the box the server is running on as the external ip, like
192.168.1.25 or whatever that address is. Then set the login uri on Hippo to
http://192.168.1.25:9000/ and it should fly.
