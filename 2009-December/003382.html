A few days ago I added a module to OpenSim that allows for more 
interesting OpenSim Libraries. For the time being it works only for 
standalones.

What it does: it allows arbitrary content to be added to the Library, 
available to all users of a world.

How: simply place IAR files under bin/Library.

Here's my blog post with more details: 
http://www.metaverseink.com/blog/?p=29

The diva distro is using this already. I've also changed the OpenSim 
official distro config files, so that this is on by default for standalones.

This works for master HEAD as of last Friday.
