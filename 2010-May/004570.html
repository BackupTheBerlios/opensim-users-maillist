This doesn't show anything unusual at all.  In these cases, the error has probably only been printed to the console - try and capture this (if you are running on Windows you'll need to start the server at a command prompt rather than by double clicking).

If you're running with Mono, then console errors are very often Mono crashes.  There's little OpenSim can do about this.  If Mono is crashing then I suggest that you try upgrading it - the realistic minimum nowadays is 2.4 - people also report good results with Mono 2.6).
