To elaborate a little on what Chris said, the initial avatar you get when
you start is running from the common library which you as a user cannot
change.

You should make the new "Shape", etc first thing whenever you set up a new
user/avatar in that user's inventory space.

This is true for any object, wearables etc. that you use from the library.

If there is not a mantis suggesting that this be done automatically somehow
as an enhancement then maybe you could start one.
