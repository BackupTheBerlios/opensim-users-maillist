Hi,

The Diva distro is intended for a simple standalone ("sandbox") type system.

It was never intended to be a "fork", only a really simple way to get
started with a standard configuration.

If you want to spread the load across a grid then the normal stable release
is the way to go - the actual coding differences are minimal, only your
config file should change.

Obviously you would then switch to running in "grid" mode in your config
file. :-)

I have not tried the new ROBUST grid setup, but the config informaiton that
you used witht he Diva distro should be enough to get you started at least.
The config.ini.example file is quite well documented.

Karen
