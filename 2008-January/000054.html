Since this just caught me out, in case you're seeing the message

Unhandled Exception: System.IO.FileNotFoundException: Could not load 
file or assembly 'ScriptEngines/DotNetEngine' or one of its 
dependencies. The system cannot find the file specified.
File name: 'ScriptEngines/DotNetEngine'

on startup, then you need to change the entry for script_engine in your 
OpenSim.ini to now point to the full dll path as shown in 
OpenSim.ini.example
