I could run opensim (SVN #7894) quite stable during the last week. 4 regions
did run for 3 days without any restart required. 4 regions needed a daily
restart and 1 region about every two days.

I run my region servers on two Intel quadcore servers with 4 GB memory under
Mono 2.3. One server uses Ubuntu 8.10 32 bit (4 regions), the other server
Ubuntu 8.04 64 bit (5 regions).

Usually there are only problems with regions that have much traffic and/or
with much building activity. Usually that increases memory consumption over
time, until the region server crashes finally. I use a tool called "monit"
that restarts the region processes automatically before that happens.
