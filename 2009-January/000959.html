Just a clarification about the interregion comms work that's going on.
Even though this is REST style over HTTP, this has nothing to do with 
the "REST plugins" that exist in OpenSim and that allow you to access 
region information from different clients. The only thing in common is, 
well, REST style, and what that means. But there is no use of the 
application plugins, at least for the time being.

So if you were using the existing REST plugins for accessing region 
information, keep on using them. If you weren't using them, keep on not 
using them. There is no need to change anything in OpenSim.ini.
