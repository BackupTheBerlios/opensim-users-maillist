Speaking of the b-day event could some of you guys who have events 
listed on the wiki please double check to make sure you're running in 
hypergrid mode and that your settings are correct?  There are quite a 
few that I've been trying to link to and I keep getting: "05:21:31 - 
[HGrid]: remote call returned an error: Requested method [link_region] 
not found" which if I'm not mistaken means that you're not running 
opensim.exe with -hypergrid=true.  I've also received an address not 
found error on a few others. 

Errors like this mean that people from the event will not be able to 
reach you via hypergrid teleport.  Also, please remember that when 
linking you need to specify the address and port of the sim you want 
people to hypergrid into.  I've seen a few that have address like 
grid.mylocation.net 8002 which if I'm not mistaken is the address and 
port of the ugaim user server and not the sim.  Most sims run in the 
9000 port range so if your sim is running on the same machine as your 
ugaim then it would be grid.mylocation.net 9000 or if on a different 
machine it would be mysim.mylocation.net 9000. 

Just a friendly reminder before we wind up with people hypergridding 
into the void come event time tomorrow! :)

Thanks, ;)
