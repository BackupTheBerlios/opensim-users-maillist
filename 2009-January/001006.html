  Last I checked friend management works, although they don't seem to 
have friend permissions (edit objects, find on map, etc) working yet.

ownership and transfer that is possible in grid mode but you need a few 
web and database hookups.  I'm using the wi-redux package from the 
opensim forge for that and so far it seems to work without a hitch.  :)
