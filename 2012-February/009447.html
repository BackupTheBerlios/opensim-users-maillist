I may be crazy, but I swear I'm seeing an immediate and very noticeable
improvement in performance from the end user perspective.  Avatar movement
not as sluggish, almost snappy even, textures rezzing more quickly..

The CPU usage is about the same as before the split (57-59% on average with
no activity) but the Page File usage is higher.  I'll be interested to see
how things go when the HGAC visits tomorrow to give a good test with higher
concurrency.
