I've been running windows 7 as an operating system has the task scheduler as
part of the operating system and windows 2008 server has as well

I use the scheduler to start my robust and opensim servers after updates or
a power failure

It can be set up to start or restart through a batch file and the timing of
the starts can be programmed into the task

I found this a lot easier than trying to set opensim up as a service and it
has worked flawlessly since I set it up

The scheduler can be used to invoke a cron instance if needed

 

http://www.windowsnetworking.com/articles_tutorials/Working-Windows-Server-2
008-Task-Scheduler-Part1.html

 

Hope this can help
