Aha! That did the trick. Apparently when using the grid manager to add 
in the grid, for some reason, it wants to change the loginuri to 
localhost in the xml file. The heck =/? lol

In any event, I found the grids.user.xml file and saw the problem 
immediately in the section for my os instance:

<key>loginuri</key>
<array>
<string>http://127.0.0.1:9000/</string>
</array>

Changed localhost to my IP address and works like a charm.

Thank you so much Wade! =)
