Hi Chris.  Information on this is patchy.  There have been some previous posts in this list but I don't think anything 
has been collected up into an OpenSimulator wiki page yet (?).  I think behaviour here will be pretty much idential on 
the Linden Lab's grid and OpenSimulator grids as all the processing work is done in the viewer.

SoftImage XSI to process and upload COLLADA models to SL/OpenSim.  I haven't tried any of this myself.
