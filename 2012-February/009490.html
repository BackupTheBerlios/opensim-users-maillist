As I was pondering in my last post (I should have read ahead), "things 
get full" is an issue.
It sounds like Mono may have a "memory leak",  which can cause available 
memory to be filled and run out of space after days or weeks.   Or, if 
it's a Microsoft operating system,  after about an hour or so. (Reboot!)

I had a long heated argument with a headstrong (dense) administrator 
once when one of my systems hung up that way.  She was convinced I had 
"done something" because she was sure that software was not like 
hardware, and if it worked yesterday, it would still work today unless I 
had messed with it.

Read all about memory leaks here:  http://en.wikipedia.org/wiki/Memory_leak
