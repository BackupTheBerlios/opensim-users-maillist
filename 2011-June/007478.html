These things are clients -- they don't know anything about Opensim
especially, the version of libomv used in Opensim doesn't matter. As
long as the protocol is compatible, and AFAIK LLUDP is unchanged for
ages at least enough to be compatible so that old clients can connect.

So I think you can use whatever version of libomv that works for you on
the client side. If you really want a bot client, instead of a bot on a
server (in which case those tutorials don't apply).
