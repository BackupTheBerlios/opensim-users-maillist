Make sure you have .NET up to date.  Win7 has .NET 4 installed, I
believe, but Vista and XP need to have it installed and updated.
OpenSim needs .NET 3.5 or better on a Windows machine. Make sure ALL
your .ini files use 127.0.0.1 instead of SYSTEMIP I'm also running the
latest SoaS, though on a Vista machine. I live in a small town and my
Internet is intermittent, yet have no troubles running up to 16
regions stand-alone. One thing I did do was to unzip SoaS into its own
folder on my 2nd harddrive, so I am not running it from a stick. I
don't think that has any effect, but you never know, which is why I
mentioned it. I also set up my shortcuts directly to Mowes.exe and to
Opensim.exe.
