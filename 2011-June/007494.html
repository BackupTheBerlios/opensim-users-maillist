Hi Wayne,

OpenSim by default uses SQLite, and stores its main database in a file
called OpenSim.db under the /bin directory. There are also three other DBs:
Asset, inventoryStore, and userprofiles.

For information regarding setting up MySQL and OpenSim, please visit
http://opensimulator.org/wiki/Mysql-config_example
