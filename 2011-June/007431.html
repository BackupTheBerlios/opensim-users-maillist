I forget the actual setting, but yes there is an adjustment. I don't
even remember if it is in the OS .ini file or in the viewer, but it is
there somewhere!

In case you didn't know the problem is that the avatar's z axis
(vertical) position is set the center point or hip with an offset,
rather than by the position of the feet etc. It is this offset that you
need to adjust.
