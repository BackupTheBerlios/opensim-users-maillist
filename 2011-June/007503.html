On 14/06/11 14:21, R. Gunther wrote:
> For some reasons i want to have some regions not on my main avatar.
> Is there some easy way to change the master avatar in the database ?
> I use mysql with seperate estate database. More regions are stored in
> the estate database.

There's no such thing as a master avatar any more, just estate owners.

At this point, you'll have to manually update the required database row in the estate_settings table so that the 
EstateOwner column contains the UUID of the new owner.
