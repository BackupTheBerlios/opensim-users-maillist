Don't use ryukyu.dnsdojo.com in GroupsServerURI.  If everything is on a standalone use localhost instead, like

     GroupsServerURI = http://localhost/XmlRpcGroupsServer/xmlrpc.php

If your router doesn't implement loopback then ryukyu.dnsdojo.com won't work.  The viewer never communicates directly 
with the groups service, only indirectly through the simulator, so there is never any need to use a viewer-accessible 
url for GroupServerURI.
