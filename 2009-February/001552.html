Yes, sorry for not e-mailing the list with this; it kind of slipped my mind.

 

I upped the Interface Version today, as we were seeing considerable
deterioration of quality of service between new and old regions. In fact,
latest regions was reported not to connect or tp at all with 0.6.2 grid
services.

 

Since interface version 2, we have done considerable changes in how regions
communicate with each others and the grid.

 

Last revision running on version 2 was 8543, so stay on that or earlier
until your UGAIM has been updated.

 

To use later revisions than that, the UGAIM, ie, the grid services, has to
be updated past that.

 

I have also applied the update to 0.6.3-post-fixes, so 0.6.3 regions cannot
connect to pre-0.6.3 grids.

 

I should also take the opportunity to invite you all on a discussion on how
we should handle interface version increases. Of course, they should
probably be made around version changes, so one indicates another. Sorry to
say, that has not happened here, so some confusion should be expected.
