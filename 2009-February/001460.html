That's just the file format extension. The file doesn't have to be named 
.oar. In fact I save all my oar files as .tar.gz. it's smaller than a ziped 
.oar.
Use the command  "load-oar path/15_02_2009.tar.gz" replace 'path' with c:/ 
or wherever you have the file saved and without the quotes.
