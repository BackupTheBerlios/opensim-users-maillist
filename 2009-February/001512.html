Hi Julia,

 

You can run OpenSim as a 32bit process happily on Windows 64 (and it
runs very well) without any recompilation fun by running
OpenSim.32BitLauncher.exe instead of OpenSim.exe. But if you want to run
as a native 64bit process, then yes, I believe there's some fun involved
with ODE that I have to admit I've not explored yet.

 

Hope that helps,

 

Chris
