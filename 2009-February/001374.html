Wayde,
Are you talking about the "welcome" option in the "GridInfo" section of 
OpenSim.ini file  ?

If yes, just update this option with the URL pointing to your apache web 
server.

According to my experience, it's only useful when you are using "Hippo 
OpenSim viewer".
Doesn't help with regular S/L Viewer. ( but I may be wrong ).

The regular S/L Viewer use the "http_loginform.html" file and the 
internal web server provided by OpenSim.

Just for your information, I use this method for more than one year :)
