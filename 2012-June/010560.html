Hello,

I am truly hoping to make this code open once I am ready. I am really
liking the results so far ...
I am really happy with how opensim runs on kitely ... Am no genius and with
my very small team, it helps to use a hosted service like Kitely.


This demo demonstrates how it may be possible to simplify content
production by users in virtual worlds. For users, content production and
the ability to manipulate objects in their virtual environment are actually
the same problem, not two. Most of the content building tools available in
current virtual world platforms that allow in-situ building is geared
towards 3D modelers.
http://youtu.be/bvQV4xN0Q9Y

There are a lot of other platforms out there ... that have tried something
of that nature ..Roblox etc.... but I think a lot more can be done actually
in opensim.

Please let me know if you can help me improve it. I don't know where to
start for developing opensource infrastructures.
