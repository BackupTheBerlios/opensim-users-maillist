Gudule Lapointe <gudule at spekuloos.be> wrote:

>Hello
>
>We wanted a way to get connection info for hypergrid users. It looks
>like this information is not saved in the database, so not usable for
>external modules like web interfaces.
>
>So we created it:
>	- a "cacheusers" table to add in robust database 
>	- a script to feed this table from robust log (with a cron batch)
>	- a php file to fetch grid info, including monthly stats for all
>visitors (local users and HG visitors)
>
>I think the table could also be used inside OpenSim to keep trace of HG
>visitors, so it could eventually be adjusted following global needs and
>added in OpenSim core.
>For example, better use would be to update it upon HG visitor login,
>instead of running a script for that.
>But for now, it is an independent project.
>
>We published the current code (along with other useful tools) at
