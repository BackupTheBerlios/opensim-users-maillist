If you're selecting users in the same region or "near" you, then NPCs will
likely appear in the user interface. Unfortunately the viewer has no way of
knowing the difference between an agent which is a user client, or a NPC,
so any user interface that chooses nearby agents will have the same
problem. This is a limitation in the protocol which has not been addressed
by Linden Lab and probably wont be as they do not have NPCs.
