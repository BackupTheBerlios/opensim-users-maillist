Oops… Didn't mention every address must be replaced by your actual server address, though it sounded obvious to me.

So, everywhere you find an url (beginning with http://), ensure the following address is your actual server address or IP.

For example http://myassetserver.com:8003/assets/ would become something like http://192.168.0.3:8003/assets/
(be sure to keep the port part)
