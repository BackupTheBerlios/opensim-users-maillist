I'm working on improving the Collada exporter by adding texture
import. I have already implemented it in Radegast which I will use as
a prototype for what it will look like in Singularity.

If you want to to check it out, download Radegast nightly build from
http://radegast.org/wiki/Radegast_Download#Nightly_Builds

Then once logged click on the Objects menu which will bring up a list
of objects nearby. Right clicking on an object there will give the
option to save as .DAE. You will see the window like:

http://i.imgur.com/TeGv4rS.png (the left part)

After importing to Unity3D it will look like this (textures and all)

http://i.imgur.com/aYGwg7K.png
