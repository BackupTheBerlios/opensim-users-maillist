> For the diva distro:
> 1) Create the other schema in MySql, with the new name
> 2) Give the opensim user the proper permissions to that schema
> 3) Run Configure.exe on the second instance
> 4) Before running that second instance, edit its file
> config-include/MyWorld.ini. Locate the connection string configuration
> variables (there are 2 in 0.7.0.2) and change the Database name to the
> new name.
> 5) Run the second instance
>
> In general, the database name is given in those connection strings in
> whatever .ini files you have there. You can set them to whatever name
> you want.
>
