OpenSim Git master now uses 6.3.6, unless you're referring to a Diva distro development version.

On 10/03/11 18:45, Diva Canto wrote:
> Indeed, there is an issue with the MySql connector (MySql.Data.dll) and the version of the MySql server that you are using.
>
> The official OpenSim tagged releases all use version 5.2.5.0 of the connector. Developer's code uses 6.2.4.0.
>
> It turns out that 5.2.5.0 does not work with the newer versions of the MySql server (5.1.55 and above). 6.2.4.0 works well.
>
> (The newer connector has some other issues, but hopefully people won't bump into those as often.)
