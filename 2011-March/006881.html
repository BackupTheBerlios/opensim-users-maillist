Ok follow this steps
1.- download xoops cube from this direct link

http://downloads.sourceforge.net/project/xoopscube/legacy/legacy%202.2.0/Package_Legacy_2_2_0_RC_2.zip?r=http%3A%2F%2Fsourceforge.net%2Fprojects%2Fxoopscube%2Ffiles%2F&ts=1300957171&use_mirror=kent

2.- Install xoops cube 
2.1 extract file Package_Legacy_2_2_0_RC_2.zip 
2.2 move html folder into your public web folder and rename it with.. "grid"
or as you like.
2.3 call http://yourwebserver/grid or html or ... as you rename in 2.2
2.4 follow steps of wizard.

When finish... 

3.- Download xoopensim module (thank you to iseky for this great work) from 
http://www.nsl.tuis.ac.jp/xoops/modules/d3downloads/index.php?page=singlefile&cid=5&lid=3
cliking on diskette icon and obtain xoopensim-1.55.tgz

3.1 untar -xvzf xoopensim-1.55.tgz
3.2 move xoopensim directory into "/modules folder of your xoops
instalation.
3.4 Login in your admin of xoops and install the module.
3.5 Configure your xoopensim acording your opensim configuration

4.- Configure your opensim as describe iseky in 
http://www.nsl.tuis.ac.jp/xoops/modules/xpwiki/?OpenSim%2FWEB%20Interface%20%28E%29%2FCommon%20Simple%20Settings

5.0 donate if you like to Xoopensim proyect for iseky work happy more when
new versions will be liberated
