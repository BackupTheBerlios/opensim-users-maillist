Right, this config file is no longer used.

Everything about mysql config is located in OpenSim.ini, and also in  
the following files if you are in Grid mode :

AssetServer_Config.xml
GridServer_Config.xml
InventoryServer_Config.xml
MessagingServer_Config.xml
UserServer_Config.xml
