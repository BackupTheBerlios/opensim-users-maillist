1. Create a brand new region in a "standalone" sim using default SQLite. 

2. Go to the existing region where textures are not loading correctly

3. In the opensim window, do "save-oar  mystuff.tar.gz"

4. Go to the region created in #1, and do "load-oar mystuff.tar.gz"
