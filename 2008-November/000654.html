Hello all,

This is mostly a question for the people  with more in-depth knowledge
of the build process and some underlying dependencies.
Let me explain how I used to run my regions on OSGrid and personal
grid (all on debian lenny/sid):
What I used to do before revision 7325 was
- copy the current revision into a dir with the revision number,
- run prebuild.sh, run nant
- move the current production directory away
- copy .svn and bin to the new empty production dir
- start OpenSim.exe in a screen session
(for my grid at home I used moo to do this, after building I push the
production directory over scp to my other machines)
