Paul,
 
there is no 'root' region - 'root' means 'all regions' for the commands that work on all loaded regions.
 
save-xml2 does not do that, so it simply picks the region that happens to be first in the list. (Internally, it uses the CurrentOrFirst method)Best regards,Stefan AnderssonTribal Media AB Join the 3d web revolution : http://tribalnet.se/
