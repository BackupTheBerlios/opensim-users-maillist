Ursula MATOVA wrote:
> Hi all,
> 
> Just a quick question,
> 
> Is there a way to define/create objects in the OpenSim default Library ?
> 
> For now, it's possible for Animations, Textures, Scripts, Notecards,  
> BodyParts, Clothing, ...
> 
> But, what about Objects, Gestures, Landmarks, ... ?
> 
> These couldn't be dumped for now ( with the "dump_assets_to_file =  
> true" OpenSim.ini option ).

The simplest course to dump these is probably to use Adam's new file 
database option to store all the assets on the filesystem on standalone 
mode.

Alternatively you could put them in a box and dump the region to an 
archive.  If you untar the tar.gz you should find them in the assets 
directory.
