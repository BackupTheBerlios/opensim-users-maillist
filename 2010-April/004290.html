default_grids.xml is the list of grids the viewer starts with. But,
usually it supplements its list from this web page when the viewer
launches:

  http://opensim-viewer.sourceforge.net/db/grids.php

So, if you put your grids in default_grids.xml, they will appear in
the viewer's list, along with the grids downloaded from that web page.

If you want to, it's also possible to tweak the viewer settings (no
recompile needed) so it doesn't download any more grids from the web,
so that *only* your grids will be on the list. We had someone ask how
to do that for Imprudence recently. The instructions should apply to
Hippo as well:

  http://imprudenceviewer.org/forums/viewtopic.php?f=6&t=359

- Jacek
