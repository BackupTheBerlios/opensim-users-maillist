For info, Penh :

I've just finished reinstalling freeswitch (1.0.6) on a new debian box,
serving different regions on OSGrid and standalones on different
port-ranges. (opensim  0.6.8 (Post_Fixes) && opensim 0.6.9-post-fixes
5ea498a).

you really have to follow the tutorials very closely. Mac/Linux/or Windows :
http://opensimulator.org/wiki/Freeswitch_Module
AND http://wiki.freeswitch.org/wiki/Installation_Guide
(especially, pay close attention to the xml_curl module, cause of most
failures)

When ready, start freeswitch before starting your regions.

Then, you might have an issue with the latest viewers interracting with the
latest opensim versions : though enabled in the estate, voice cannot be
configured in the "about land"->media tab for your region (only greyed out
option boxes). Quickest way i found was to reinstall meerkat, which is an
"old" viewer and enabled me to access the "enable estate/private channel"
option boxes. Then back to the more recent viewers.
NB : if you use imprudence, don't forget to install slvoice too (it's not in
the packages anymore for political reasons).
