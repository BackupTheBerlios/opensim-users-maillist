You need to setup and install the flotsam XmlRpc service that is written 
in PHP, or you have to use Simian Grid Services.

The XmlRpc service is available from:

http://github.com/mcortez/flotsam/tree/master/flotsam/Servers/XmlRpcGroupsServer/

You have to setup the database manually using the included SQL.

Current OpenSim/master/HEAD as well as the stable/post-fixes branch work 
fine with this service.
