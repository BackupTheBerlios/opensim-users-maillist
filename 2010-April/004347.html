Yes i do, in any viewer ..

default
{
    touch_start(integer total_number)
    {
        llLoadURL(llDetectedKey(0), "Now go to: ", "http://osgrid.org");
    }
} 

the URL given must begin with "http://" or "https://"; it will not work with
other protocols 
such as ftp, telnet etc. If a user mutes an object, they will not receive
any URL offers from that object.
