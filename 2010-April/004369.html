Hi Jason,

Freeswitch points to the simulator's http listen port(default is port 9000) 
that has the freeswitch module enabled. So yea, probably 9000, unless you've 
changed that(also in gridmode). Do you know about this tutorial? 
http://maxping.org/resources/how-to/install/freeswitch-on-ubunutu.aspx

It's for linux, but i guess most of the options are the same for windows as 
well. Also make sure you have the SLVoice.exe in the viewer. Not all viewers 
have that by default(hippo has, but imprudence doesn't). Also, if voice is 
important to you, check out the realxtend project. It has mumble integration 
by now. Voice quality is much better than with the proprietary slvoice, but 
the naali viewer is not completely stable yet. 

Jeroen
