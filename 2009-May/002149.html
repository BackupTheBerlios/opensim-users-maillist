.. do you as well hate it accidentially shutdown your whole sims by clicking

on the X Button of the WINDOWS DOS console window ?

Me too and therefore i was looking for solution. I found a Tool called 
AppScapeUtility.exe that is used by Citrix Administrators as well.

You have to install this utility and after a restart you can enable
it in the Windows System Control Panel. Just select that all CMD.EXE
Windows are protected.

If you start your Regions via a shortcut eon your desktop e.g.

C:\WINDOWS\system32\cmd.exe /k "H:\opensim\OpenSim.exe" 

the X Symbol in the top Right corner is greyed out. If you SHUTDOWN
your regions you can close the DOS Window with EXIT as usual.
