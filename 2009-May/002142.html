Hi Robert

 

This problem can happen when you use the ODE engine with meshing =
ZeroMesher. If you use the ODE engine you should comment this parameter and
enable Meshmerizer in OpenSim.ini:

 

    ;meshing = ZeroMesher

    meshing = Meshmerizer

 

Cheers

 

Volker
