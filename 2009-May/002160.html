Hi Marian,

 

ReactionGrid is spread over several servers, so you will get different
IP address returned for different sims - and a grid like OSGrid would
have many different IP addresses since anyone can connect a sim to the
main grid. I would suggest the Shard header from the http request, if
set by the grid operator as detailed in opensim.ini.example, is probably
the best way to go. Note that current revision deployed on ReactionGrid
doesn't seem to pick up this value from the ini, but tested against a
newer release it does seem to set correctly, so will hopefully be
available on our grid in the next week or two.
