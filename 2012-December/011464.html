The only thing you do in the bin directory is add the paypal template html
file in the bin folder and add the information contained in README.txt into
your OpenSim.ini file.  Be sure to add the paypal.ini file to your
config-includes folder also.

The only thing confusing about snoopy's readme file that I can see, is
whether it allows users to exchange currency inworld without going through
paypal directly each time.

Hope that helps a little.
