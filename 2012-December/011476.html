If using flotsam you will need to have a php server (wamp or xamp work well)
and need to have xmlrpc enabled in the server.  Also if your opensim is
meant for grid mode your better off using mysql especially when it comes to
groups ive found.  

Also be sure you have your OpenSim.ini  [Groups] section configured properly
to point to your groups server.  I have groups working though with some
minor errors otherwise it works.

I hope this helps a little bit.

Emperor
