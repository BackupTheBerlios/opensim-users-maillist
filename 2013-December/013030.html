Hi Chris,

you can cross prims using all manner of scripts. Whether script
state or even code can cross is determined by other settings that
haven't changed. Also llSetKeyframeMotion will cross prims.

Prims cannot cross the hypergrid because HG is implemented as a
teleport rather than a crossing.

- Melanie
