That slideshow is not a good reference for installing a regular networked
opensim standalone, those slides were for a faculty workshop I gave to show
them how to create a private completely NOT-networked sandbox region.

If you are just getting started with Opensim, my suggestion is to start
with a Diva Distro pre-configured standalone installation, which you can
download from:

https://github.com/diva/d2/downloads

You probably want the third link called "Preconfigured hypergrided
standalone (OpenSim 0.7.3.1)".

After you unzip the file, read the Installation instructions and it is
pretty straightforward.  The Diva Distro also comes with the Diva Wifi gui
module pre-configured so you don't have to do a separate installation.
