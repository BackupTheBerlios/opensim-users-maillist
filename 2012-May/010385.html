Then you have a 64bit system. Memeory over 4GB can not be addressed by a 
32 bit system as there is a limit to how much memory a 32 bit system can 
address before it is unable to address it for use. Now the motherboard 
or the CPU may be reporting to windows that there is 6GB available but 
unless it is a 64 bit system the entire 6 GB will not be available.
