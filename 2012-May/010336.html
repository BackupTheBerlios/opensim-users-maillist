Hi P, the best package to use IMO is Mimetic Core's Opensim services bundle
- it has profiles, search, offline IM, and groups all in one nice tidy
package.  You can download the version for 0.7.2 from
http://opensim.odosys.net/files/

I found it much easier to install the bundle than trying to install each
piece separately.

Good luck!
