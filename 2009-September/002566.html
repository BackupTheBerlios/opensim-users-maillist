Also, to make life simpler to start, I suggest the following after
downloading and starting L3DT:

1. File -> new project (select Designable map)
2. Set width and height. For example 512 for both. This will
    require 4 opensim regions in a 2x2 configuration
3. Take default HF/DM ratio
4. Modify the Design map parameters to whatever you want--
    more sea? rolling vs. fractal, etc.
5. click on Heightfield in the next window and then OK
6. Operations -> Heightfield -> Change vert. scale
7. Try something like Min alt of -20 and Max alt of 120 to start, and 
press OK
8. File-> Export -> Export Map
9. Choose Heightfield
10. For file format, scroll to R32
11. Hit the browse button and enter a name such as myterrain.r32
12. Assuming you have set up 4 contiguous regions in a 2x2 configuration 
in opensim...
13. And that you have added a new line "CombineContiguousRegions=true" 
to [Startup]
      in OpenSim.ini
14. copy myterrain.r32 to your opensim\bin folder
15. launch opensim and stay in the (root) region. you should be there by 
default
16. Fly to a high location, say 160 meters up
17. do "terrain load-tile myterrain.r32 2 2 1000 1000" in the opensim 
console
18. Enjoy smooth motion throughout the space

Let me know if this works -- took a bit of twiddling to get all the 
steps right
