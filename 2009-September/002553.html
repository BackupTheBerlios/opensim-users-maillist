Well, I can tell you the following, from your mails I have tried to render a
webpage locally and it appears to successfully crash the Hippo viewer... No
dynamic webpage, just one of the most basic html pages out there and
unfortunately it doesn't deliver. As for the root cause of this, in previous
responses there can be seen that all that the GridInfo section in the
OpenSim.ini file does, is point towards the correct file. This is all neet
and dandy BUT, this will point to a LOCAL file. So in either way I now
assume that rendering a html page on the system pointing towards a relative
(local) file is not possible, simply because the user will not have that
local file on its local file system. As said before, install apache ;-)

Cheers,

Ben
