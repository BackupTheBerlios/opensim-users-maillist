You need to make the regions start in a specific order
South to North, West to East.
Order them differently in Regions.ini or name the files in alphabetical
order if you are using .xml files still

It only works on the most recent development, bleeding edge, versions of
OpenSimulator in the Git repository or the SVN mirror.   There are no
release downloads that you can download yet with this feature in.
