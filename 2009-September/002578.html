Hi,

I haven't seen your mail in the realXtend mailing list about this issue, did you sent the mail in correct address? However if you are using the 0.5 server check the following config under [realXtend] section in OpenSim.ini:

 ;With this property you can set maximum height that avatars can fly to in region.
 ;Zero when disabled
 FlightCeilingHeight = 0
And if you are using 0.4 then the appropriate configutation is located in xml-file in regions folder.
  flight_ceiling_height="1024.0"I hope this helps.
