Hi,

Just a note to let you know that as of this afternoon (r10869), the 
simulator has been switched to interact with the new-style grid service.

People running standalones don't need to do anything about this.

People running grid servers should now run the grid service inside a 
ROBUST server shell. Please see:
http://opensimulator.org/wiki/ROBUST
and in particular
http://opensimulator.org/wiki/ROBUST#An_Example_Conversion_From_UGRM_To_URM

Cheers,
