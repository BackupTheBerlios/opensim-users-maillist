Hi everyone

I seem to have found the solution to the T position frozen avi problem :)

I could log in but my avi was left frozen.

It turned out eventually to be an issue with our institutional ISA
firewall that was dropping the UDP connection on port 9000 between
client and server after login.

OpenSim on the vleinternal server is working fine now the firewall
tunnel settings have been amended.

Best wishes

Clive
