Hi,
The log config file is OpenSim.exe.config or OpenSim.32BitLaunch.exe.config
if you are running the OpenSim.32BitLaunch.exe. Every executable has their
own log4net configurations.

The -logfile option doesn't specify the configuration file, but
it specifies the file to which save the log.

Mikko
