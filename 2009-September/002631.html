Try turning on beacons in your client for objects with active scripts -
and if it's a physical object, view physical objects too. You should see
red lines for scripted objects, and a green line for anything physical,
which should help you locate it. Also check the Region/Estate menu's
debug tab and look for top scripts in the list and highlight it if you
find it.

 

If the client fails you it's possible to extract from the database if
you know where to look (equally possible to cause harm if you don't know
what you're doing), so suggest you try inworld first and see how you get
on.
