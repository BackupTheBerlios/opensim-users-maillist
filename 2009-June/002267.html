Right on.  That is where the cache is stored by default.  That can be
changed in the application BUT those settings are all stored as registry
settings.  Those settings are what I'd like to have as an ini file rather
than registry settings.

For example, if you go into Second Life and change the cache to be stored on
a thumb drive along with the main application, if you plug the thumb into a
different computer and run Second Life, all the settings revert to default,
including the default cache storage location you pointed out.
