I've also successfully used WebThumb (http://webthumb.bluga.net/home) which
gives you 1000 free captures.  However, ultimately if you this kind of
functionality is a core part of your OpenSim environment, you should think
about hosting your own HTML-image service, using something like CutyCapt
http://cutycapt.sourceforge.net/ - we've done this and it makes you feel
much happier to know that you control the service!

Other alternatives are possible too - if you have a limited number of
visitors to your region that you have control over, you can give them
instructions on modifying their viewer setup so that they can see 'real'
parcel media HTML-on-a-prim.  Alternatively move to 0.7 if you haven't
already and start exploring Viewer 2 and the new Media-on-a-prim
functionality :-)
