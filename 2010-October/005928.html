Hi Harri,

1- Create an account
2- Login in-world http://myopenGRID.com
3- Create a box put your items inside (t-shirt etc...) thake the box back to
your inventory
4- Access the marketplace https://myopenSIM.com (you have access to your
inventory there)
5- Set the box to sell (add pictures, description etc) done.

The buyer will receive the item directly in his/her inventory.

Easy :) if you have more questions try support at myopengrid.com
