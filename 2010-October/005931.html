Hi,

Unfortunately ModreX code hasn't been updated for some time with OpenSim
master, so it will be most likely broken on some bits. RexOpenDynamicsEngine
is inside the ModreX repository and is built like the other parts of ModreX.
Patches on making ModreX work again on OpenSim master are highly
appreciated. If you need help on this, we can discuss this on realXtend
mailing list (http://groups.google.com/group/realxtend) or #realxtend irc
channel on freenode.

Cheers,
Mikko
