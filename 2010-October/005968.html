i downloaded the binaries from
http://wiki.realxtend.org/index.php/Getting_Started_with_Taiga  
i've used the Wizzard to setup everything, but its not full proof. you still
need to go trough all ini files...

bin dir
----------------
OpenSim.ini
OpenSim.Server.ini


bin/addon-modules dir
-----------------------
modrex.ini
mumblevoice.ini              (if you are using this)


config-include dir
---------------------
CenomeCache.ini
FlotsamCache.ini
Grid.ini                         (for grid only)
GridCommon.ini
GridHypergrid.ini             (for HG only)
Standalone.ini
StandaloneCommon.ini
StandaloneHypergrid.ini  (for HG only)

Regions dir
-------------------
Regions.ini


make sure all points to 127.0.0.1 with correct port numbers, 
with mysql, all databases exist (before you start).  after that, create a
user. (change in Regions.ini)
