There's a 32 bit launcher included with OpenSimulator that will run
OpenSimulator in 32 bit mode on your 64 bit machine in Windows.  The
alternative is to recompile the native libraries.   In Linux, mono has a
system for automatically choosing which platform native library to use based
on your hardware.
