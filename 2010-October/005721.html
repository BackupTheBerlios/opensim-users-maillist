Are you running OpenSim in grid mode?  If not, then running ROBUST isn't necessary.

If you've never run OpenSim before I recommend getting it going in standalone mode first.

If you are intending to run in grid mode, then this actually looks to me like a failure of the region to communicate 
with the backend service that handles estate information.
