Hi Tomas,
Try the diva distribution from OpenSim downloads. It has a set of default
male and female avatars with some custom clothing. you might want to try
some of those. As for the avatar being a particle cloud, it could be because
of one of the following reasons:
1. You're using Second Life viewer 2. If so, try using viewer 1.23
2. You're using a very old version of OpenSim on a machine without graphics
card

Please provide further details on your enviroment.
