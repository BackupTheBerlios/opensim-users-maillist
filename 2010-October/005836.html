Hi Jor3l,

I currently have it running on 0.7.0.2, but also had it running on 0.6.9. and 0.7.0.1 previously.
Probably the most important point for linux is that you really need to compile the trunk version of freeswitch- the binary version just didn't work for me and judging from other lists/blogs I've read, it's been the same experience for others. You may get a few compile errors but usually it is because it is missing some dependencies. Work through those and you'll get it compiled, although if you follow the post from the link I gave, it should be alright.
