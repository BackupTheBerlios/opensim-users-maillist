Tomas,

You need to use the "change region XXX" command to change to the region you
want to act on, before before using the "terrain load path/to/texture.png"
command. Take a look at the wiki page at
http://opensimulator.org/wiki/Detailed_cross-region_terrain_making for some
more information.

Brooks
