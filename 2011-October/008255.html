I created (for OpenSim) an alpha mask like the SL invisible avatar.
It's free to be downloaded from:

http://www.magalaxie.com/upl/InvAvatar.iar

Put it in the folder Body Parts using Viewer 3 (in the OpenSim console :
load iar YourFirstYour Last / YourPswd thePathOfTheFileOnYourDisk)

Example to use it with the mesh Simple_Bot_001:

Rigth Clik your Avatar, Take Off, Clothes, All Clothes
Rigth Clik your Avatar, Take Off, Detach All
In the Inventory, right click Invisible Avatar, Wear
In The Inventory, right click Simple_Bot_001, Wear
if you need it, you can attach some accessories :)
