> issues have been submitted."

> Does this mean a get from SVN for opensim will not get me the group code

The XmlRpcGroups module for region servers is in the OpenSim SVN repo.
Think of http://code.google.com/p/flotsam as a forge.opensimulator.org
project only hosted elsewhere... stuff gets
developed/tested/fixed/broken there, and things which make sense to
have in core OpenSim are contributed back as patches.

If you want to run your own XMLRPC service for use with that region
module, though, you'll find that at the gcode site above, not in the
OpenSim SVN repo.
