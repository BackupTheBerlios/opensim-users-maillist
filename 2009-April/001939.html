Personally I agree with what Justin was saying in IRC yesterday about it would be better for the group support in core to be done as a community/core project, there is so much that can be done with groups differently to how the standard SL system does them. So it would be good if we did the designing on the mailing lists and in irc and work out what we actually want and could do. 

I think this is a good time to start to think about some next steps in opensim features and how we can expand some things while keeping to the protocols. So much more we can do than just clone SL features. Not just in groups but in a wide range of areas.
