Hi everyone,

A heads-up that starting in r9321 the configuration variable hypergrid 
moved from the command line to OpenSim.ini -- see OpenSim.ini.example.

This is the first of more changes ahead for configuring how to deal with 
the various aspects of the HyperGrid -- hyperlinks, foreign visitors, 
security, etc. Soon the single variable hypergrid will be replaced by 3 
or 4 configuration variables that can be assigned values independently.
