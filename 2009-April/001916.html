Hypergrid and inventory...

I really like the concept of hypergridding. One useful thing would be having
sandboxes off the main grid with huge libraries of components for building.
One could go there, build, then take the object into inventory and travel
back to their home grid with the newly created object.

This would help alleviate the asset server load a great deal on the main
grid, and the sandbox asset server could be cleaned on a regular basis and
the regions reloaded by oar.
