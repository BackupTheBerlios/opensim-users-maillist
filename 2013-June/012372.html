Hi Austin,

While I similarly cannot use the world map when I make a hypergrid jump,
happily, the non working map feature is limited to mesh-enabled viewers.
In turn, as a "work around," I use the non-mesh-enabled Imprudence viewer
to hypergrid jump -- so I can continue to use the world map to navigate
around in my hypergrid travels.  In other words, I think the problem is
related to the move from viewer version 1 to viewer versions 2 and 3.

Note:  The mesh-enabled viewer map function still works to locate an island
when hypergrid traveling -- IF you know the name of the island you would
like to visit.

Best,
Kay
