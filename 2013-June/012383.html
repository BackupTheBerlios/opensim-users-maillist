Mumble was previously working with OpenSimulator not that long ago - if it is not right now then that's probably because 
the de facto interfaces have changed slightly and it would not be much effort to get it compiling again.

The client side is quite static as well, so I believe the Murmur client side executable is still a drop-in replacement 
for SLVoice.exe

The most vexing issue for me is that the Mumble module was released under GPL, which stops it being bundled with the 
OpenSimulator distribution which does not use a copyleft license (being BSD).  I would love to see a BSD licensed 
version produced.  Of course, this doesn't stop it being bundled with some downstream distribution where it's easier to 
mix licenses (as the core coding isn't being done there) but this is still awkward.
