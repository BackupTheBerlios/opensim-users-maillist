It would be helpful if you told us which group implementation you
are using. The (current) standard OpenSim-version does not support
groups. If you are the DIVA-distribution, it is (currently?) limited in
the number of groups and roles you can define. If you are using the
Flotsam-modules, make sure you are using a version suitable for your
OpenSim-version. If you are using another implmentation I am honestly
not sure what is wrong, since I am probably not familiar with it.
