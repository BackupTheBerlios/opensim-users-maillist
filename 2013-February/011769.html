Hi Camila.

If you're using parcel media (sloodle 1):
- There will only ever be one URL per land parcel.
- All avatars standing on the same land parcel will see content from
the same URL.
- What matters is the land parcel you're standing on, not the land
parcel you're looking at. This is a bit strange. So if I'm standing on
Parcel A looking at a screen on Parcel B next door, I will see the
content from Parcel A, not Parcel B.

If you're using shared media (sloodle 2):
- Each screen can have its own URL.
- All avatars will see the URL for the screen. Land parcels don't
matter, and it doesn't matter where you're standing.
- If you have a lot of screens visible in the same space, some of them
may just look like white squares some of the time, because Shared
Media is a bit rubbish.

In both cases, people share the URL, but they may not always see the
same thing. For example, if they're looking a site that needs you to
log in, they may see different things if they're logged in as
different people.
