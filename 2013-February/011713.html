> im not able to use google fonts with a media on a prim website.
> is there a way to make them work ?

I found the solution was to ditch the regular Google font loader and
specify the font yourself. The issue seems to be that when you use the
font loader it tries to give you the font from "local(yourfont)" first
and only pull it from the web if it's not already there, but that
causes the whole thing to blow up with media on a prim.

So this fails:
http://fonts.googleapis.com/css?family=Candal

As does the contents of that:
@font-face {
  font-family: 'Candal';
  font-style: normal;
  font-weight: 400;
  src: local('Candal'),
url(http://themes.googleusercontent.com/static/fonts/candal/v3/c4OCAZZ9bgJkVvcf-E79mQ.woff)
format('woff');
}

...but this is OK:
@font-face {font-family: Candal; src: url(font/Candal.ttf); }
