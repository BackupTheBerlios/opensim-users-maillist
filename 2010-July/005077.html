Hi folks.  As part of my latent mascohism ;). I spent some time today updating the OpenSimulator configuration page

http://opensimulator.org/wiki/Configuration

so that the grid setup instructions reflect the changes in OpenSim 0.7.  I wrote these instructions while I set it up 
from scratch (using the 0.7 binary packages) myself, so hopefully they should be pretty accurate.  There are a total of 
11 steps, 3 for the region simulator and 8 for the ROBUST services (in pure default configuration).

Of course, any improvements are very welcome - just go in and edit the wiki or put something on the discussion tab of 
that page.

I also included on that page some colorful diagrams of standalone and a very simple grid configuration, showing where 
the services are located in each.  The source files for the diagrams are also linked there.

Finally, if anybody is interested, there's a little bit more discursive explanation of these diagrams and some 
architectural ramblings on a new blog post at

http://justincc.org/blog/2010/07/30/pretty-opensim-07-diagrams-and-updated-grid-setup-instructions/

Enjoy!
