I found some solid looking info about connecting Freeswitch through a
firewall on this  http://wiki.freeswitch.org/wiki/Firewall Freeswitch wiki .
I'm away from work for the next week so I can't sit down with the network
admin to tweak what we've already done. But I can see that I don't yet have
the ports used for SIP connections set right. I think that will be key and
may fix my problem. One other thing I found is in opensim.ini under
freeswitch section if I set only the "external service IP" to the opensim
server's external ip and the rest following in that section to the local IP
I can still connect to freeswitch and get service locally, otherwise with
all IP's set to the external I couldn't. So much partial and misleading info
out there about this!
