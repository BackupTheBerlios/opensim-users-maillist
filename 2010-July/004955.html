> I get this error too (System.Xml.XmlException: Root element is missing).

Zane, it won't be related to Mantis 4871 since that's to do with region xml and this trace suggests that the user 
service on a 0.6.9 grid is not returning the requested data.  If your problem is due to mantis 4871 then I suggest you 
try the latest 0.6.9-post-fixes since there is a backported fix there for it.

In this case, Peter, I'm going to guess that the power outage corrupted your user database.  You may need to restore it 
from a backup.
