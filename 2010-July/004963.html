whell you could automate it  if each sim has a name.  
like this 

change region Region 1
save oar C:\Users\<yourname>\Downloads\Backups\Region-1.oar
change region Region 2
save oar C:\Users\<yourname\Downloads\Backups\Region-2.oar
change region Recgion 3
(etc.)
(etc.)


You only have make this once. Put this in a .txt file (something like
backup_region.txt) and copy this to your Opensim folder. 
Then with the command command-script backup_region.txt everything will be
backed up. For loading works the same, 
change save oar to load oar and put in new .txt file and run it with
command-script command. copy .txt files to backup to (for save keeping)

hope this helps..
