2010/7/11 LaeMing <laeming at exemail.com.au>

> I use mysql-admin installable from the repositories to access the db for
> maintenance.
>
> Within the UI, go to 'Catalogues' in the sidebar then 'opensim' below that.
> All the opensim tables will then appear on the right.
>
> *** I STRONGLY recommend using the 'Backup' sidebar option to create a
> backup
> of the opensim database before making any changes. ***
>
> Actually, regular backups of the db are always a good idea anyway,
> especially
> if you are using alpha software (which OS is).
>
>
> There is also kmysqladmin in the repos. However, while I am a KDE user, I
> couldn't get my head around the UI of this one so stuck with the above.
