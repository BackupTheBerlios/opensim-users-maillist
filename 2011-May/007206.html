I want to write an external service which manages all rooms of an 
organisation (just a proof of concept). It should be possible to connect 
also other virtual worlds or also a real world room management tool. The 
management service has to be able to send a room configuration to the 
OpenSim region module at a scheduled time (room reservation) and the 
region module has to send the content or position of the objects to the 
management service. It should be possible to save the state of the 
learning session for later continuation. Therefore I need a two way 
communication.
