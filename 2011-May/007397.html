Hi folks.  OpenSimulator 0.7.1.1 has just been released and is available for download.

This is a minor point release on 0.7.1 and is intended to fix some bugs and cull some log verbosity.  However, I 
naughtily also sneaked in the new inventory links code which fixes problems with outfits on viewer 2 and related third 
party viewers.

There are no configuration or database changes from OpenSimulator 0.7.1.

Release notes are at http://opensimulator.org/wiki/0.7.1.1_Release

As usual, thanks to everyone who helped out!
