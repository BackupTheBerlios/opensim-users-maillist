here's my way:

if you want to download the latest opensim git to (for example) to
Downloads, 
rightclick on downloads and choose "git bash"  this will open git command
prompt.
http://opensim-users.2152040.n2.nabble.com/file/n6399747/git_shell.jpg 

then give command "git clone git://opensimulator.org/git/opensim
OPENSIM-GIT" 
this will place opensimulator  within OPENSIM-GIT folder  within Downloads.  
Optional you can give command:  "git log --since="2010" > gitlog.rtf"  to
down the latest comments.  
The next time rightclick on "OPENSIM-GIT" and choose "git bash" this will
open git command prompt again 
and give command "git pull" to update opensim repository.
