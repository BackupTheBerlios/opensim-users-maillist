There's no intrinsic 'cpu clamping' in OpenSim but this is actually a pretty complex question.  The threading that 
OpenSim uses varies a lot by area and configuration.  In this case, I would guess that the script engine is limiting the 
number of threads that is uses (maybe 2) and so a tight script loop only takes up one or a few cores.
