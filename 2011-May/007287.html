>Are there a lot of default settings now built into OpenSim 0.7.1 so they are
>no longer needed in OpenSim.ini?


Clive, in 0.7.1 there are a lot of lesser customized settings now in 
the file OpenSimDefaults.ini which is loaded ahead of 
OpenSim.ini.  You can change OpenSim.ini based on OpenSim.ini.example 
and also copy out and override less commonly used parameters by 
copying sections over from OpenSimDefaults.ini
