A couple of months ago I posted a message asking for help on how to play video and mp3 files in my regions (and not just in the SW region) while using the Diva distro.
After many trials and errors and mostly by chance, I was able to do so by doing the following:

1.       Create different regions as part of different estates

2.       Have as many RegionConfig.ini files as my regions

3.       Change CombineContiguousRegions to true in \bin\config-includeDivaPreferences.ini file (I think that did the trick)
