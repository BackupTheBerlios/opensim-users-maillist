oh man. This release is full of hickups. This was the result of an 
undocumented configuration change. I fixed the release notes for 0.7.3, 
and made a new d2 release diva-r18222-c with that fixed. Hopefully, 
third time is the charm.

The fix is very easy, you don't need to download the new package again. 
Just edit config-include/DivaPreferences.ini and in the
[GridService] section add
HypergridLinker = true

That's all I did for the -c package.

Thanks for reporting.

Diva
