Hi Wade,

Ja, hope the solution works with your model. I think so. Because SL Viewer
perfectly supports the dae model if only you use jpg image (probably other
image formats as well, but I haven't tested it). Don't forget to change the
filenames in the dae file. And you can enjoy a lot of model from the
SketchUp Online Library.
