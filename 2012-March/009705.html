I don't know what's going wrong here - there shouldn't be any config changes that could possibly strand you at 0,0,0

I'm presuming that you're using the released 0.7.3.  There was a bug in master for a short while after the release which 
would stop you moving (but that wouldn't put you at (0,0)).

A small possibility is that stronger en_US locale enforcement (all values read and written with dots for decimal places 
rather than commas, etc.) is causing this problem.  Unfortunately, if this is the case there's not much that I can 
suggest short of wiping your GridUser table which will reset your home/last location positions.  But if you're also 
seeing teleport failures then this won't be the issue.

As M.E Verhagen said, it's better to create fresh ini files on a release and copy over selected parts of previous 
release files.  Simply copying over files from an older release has caused problems in the past.
