Robert:
 From our experiments, there seems to be a bug of some sort in the way that
some textures are downloaded from the OpenSim (asset) database to the 
viewer. For
a relatively small number of textures, these textures are not fully 
rendered, however,
we have verified that those textures are high resolution in the OAR 
file. Typically,
there is a "Loading ..." text over the texture (in Edit mode) that 
refuses to fully render.
paul
