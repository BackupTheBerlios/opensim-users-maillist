You wouldn't be able to trigger a File Open dialog box without some extremely heavy viewer hacking and some custom 
OpenSimulator code.

One alternative may be to use media-on-a-prim (shared media) to display an external web page and then use that to 
trigger a local file selection.  The page could then communicate this back to a script via llRequestUrl() or similar.
