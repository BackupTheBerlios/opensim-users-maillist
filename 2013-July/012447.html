Hi Kay,

The in-world attendee tickets are currently sold out, but we still have
spots reserved for sponsors, speakers, and volunteers, so if someone is
interested in helping with the conference they can still get an in-world
ticket.  We have more information on the conference website at
http://conference.opensimulator.org or they can feel free to contact me
off-list.

For those who just want to attend, they can still register for streaming
tickets from the website.  The developers are doing lots of load testing
and bug squishing right now to improve the performance of the conference
grid, and if we think we can accommodate more folks in-world based on the
load tests, we'll convert the streaming tickets to in-world tickets on a
first-come-first-served basis.

So, I'd still encourage people to register for the streaming tickets in
case we have more room, and if they can, help with the load tests on
Tuesdays at Noon PDT so we can squish more bugs!  :)

Hope that helps but please let me know if you have any other questions.

Thanks!
