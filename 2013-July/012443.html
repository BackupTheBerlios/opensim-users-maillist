Hi Azeem,

Your file structure should look something like:

-opensim_directory
  - addon-modules
  - bin
  - doc
  - share
  - ThirdParty
  - ThirdPartyLicenses

The IAR file will by default save to the /bin directory of your opensim
installation.

Hope that helps and I'll add this information to the wiki page.
