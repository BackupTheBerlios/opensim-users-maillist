In Singularity you could go into the grid manager and delete all the
unwanted grids, but the user can still add new ones. If you're talking
about simplifying it for, erm, less intuitive users, that should work just
fine. In Singularity you can also set whatever grid you want as default, so
it will pop up first. That would also work for that target group while
keeping functionality for more advanced users. iirc the grid manager acts
just a little differently in this matter in the different viewers.
