You can also set up a "telehub" on a region and allow anyone to "set 
home here" at that point..

http://wiki.secondlife.com/wiki/Viewerhelp:Telehub

There are three types of land you can set as your home location:

* Land you own
* Land set to group access for a group to which you belong, if you 
have the necessary ability.
* Telehubs
