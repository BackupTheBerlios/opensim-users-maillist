Did the same thing once upon a time.  If it is a standalone that you don't need to access off the machine then change that to localhost or 127.0.0.1.  If you need it on the same subnet then use your IPV4 address of the machine.  If you want it on the net at large then it has to be a real externally reachable DNS name like myexample.com.
