This is pretty easily accomplished with a specific set of tools and
techniques. Essentially, you run a webserver that acts as an intermediary
on the backend for a database service (say, using php for example), with
which your lsl scripted object(s) communicate via an xmlrpc-type mechanism
implemented on  llHttpRequest () and related lsl instructions.
I can provide you with an operating example of this tech if you are
interested.
