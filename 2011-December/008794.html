I think Garmin may well be right about the bent legs issue.

However, the popping up is probably an ODE physics issue.  ODE surrounds each avatar with an invisible capsule as a 
simple approximation for physics purposes.  I can imagine this if many avatars in a small space one of these capsules 
might get squeezed up so it rests on top of the others.
