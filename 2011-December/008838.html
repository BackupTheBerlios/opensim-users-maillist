On 12/12/2011 8:23 PM, paul emery wrote:
> What is the general way to set windlight values so that they will be used by
> clients when they have the client windlight setting turned on?

So far as I know, server side Windlight settings are most easily set 
using LightShare scripting functions. These persist to the database, and 
are sent to compatible viewers. You might want to take a look at the 
following wiki page: http://opensimulator.org/wiki/LightShare
