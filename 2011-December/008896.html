If you are running your web server on a port other than port 80 it may 
help to specify that port in your URL.

Example, if your web server listens on port 4509, then you would want to 
put in your ProfileURL

ProfileURL = http://localhost:4509/os_services/profiles/profile.php 
<http://localhost/os_services/profiles/profile.php>

If that does not help you could try specifying 127.0.0.1 instead of locahost

Hope this helps :)
