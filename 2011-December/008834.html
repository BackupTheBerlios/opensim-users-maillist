We all started one day and we are all learning all the time ... It costs nothing to say that but it does not help you much :)))

Maybe you should go to a sandbox and play with prims, communications between prims and LSL.
The problem with tools like the LSL is that sometimes there are ready-made solutions and sometimes we must make ourselves.

If I tell you that there may be another solution would be to read the PowerPoint file with a PHP program to generate the images and display them on a prim with Viewer 2/3 using an HTTP request, I guess you will not be more advanced. Using media-on-a-prim, you can display a web site on a face of a prim; if you can embed your presentation on an object flash for example, there is one chance in two to display it in OpenSim.

Anyway if you want to study the LSL, take a look at these wikis:

http://wiki.secondlife.com/wiki/LSL_Portal
http://lslwiki.net/lslwiki/wakka.php?wakka=HomePage
http://wiki.secondlife.com/wiki/Category:LSL_Library
http://community.secondlife.com/t5/Scripting-Library/bd-p/2122
http://wiki.secondlife.com/wiki/Old_forum_Scripting_Library_index
http://www.free-lsl-scripts.com/cgi/freescripts.plx
etc etc

And last you can try to find among users of OpenSim and/or Second Life, a good programmer able to find a solution.
Good luck.
