Where does it say that?  As far as I know there's no such message in the Whisper module on the OpenSim side.

I really wouldn't bother with trying to set up ICE until you've got an initial setup working.  ICE just encrypts the 
communication protocol between region module and murmur server.  Although you need to have the Ice.dll present to get 
the region module to operate, it won't actually be used if it isn't configured.
