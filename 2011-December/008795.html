On 07/12/11 15:51, Rick Anderson wrote:
> Hi All,
>
> Does anyone know if the crash log feature works? I have it configured,
> but the directory never contains a crash log.

I believe this works but I've never tried it.

It's worth remembering that crashes which cause a stack trace (e.g. ODE crashes) will never make it into such a log 
since they go wrong at the mono level rather than at OpenSim's managed code level.  These won't be in OpenSim.log 
either, so the only way to record them is to log the entire console session in some way.
