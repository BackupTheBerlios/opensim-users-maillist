Unfortunately, this is not currently easy to do, at least within vanilla OpenSimulator.  Having said that, the 
UserAccountService class in the OpenSim.Services.UserAccountService package contains all the mechanisms needed to 
prewear body parts/clothing for the minimum viable ruth (skin, eyes, body, hair, shirt, pants).

Unfortunately, this is not currently customizable.  However, if you have no interest in the default ruth and only need 
one default avatar, then you might be able to hack the code to load your own assets rather than the ruth ones (which 
come by way of the OpenSim library).

There is also a similar mechanism in XmlRpcAdmin which is more customizable but wasn't suitable for that particular 
purpose at the time.  However, I don't know how well that works at present or how well it is documented.

As others have suggested, I believe that projects such as the Diva Distro have a mechanism for choosing avatars at 
creation time in the web interface.
