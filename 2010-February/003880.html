Ok guys.. I have the demo working stable now..  you can get it on my website
on my front page. Just click the OST button. AGAIN its a demo and a Win32
file. My MAC guy who happens to be a Beta Tester for me and i went over this
and without changing any settings can create the raw INI file without
crashing the sim. So if you loose your original OpenSim.ini file this will
reacreate it for you.

This will also allow you to choose MySQL and setup its info as well as the
URL domain name and a few other goodies.. BUT.. again, its DEMO .. (( Alpha
stage )) not production level yet. I'll be adding a FEW other goodies over
time, like storing your previous settings so that if you want to reacreat
the file again but without having to change everything all over again you'll
be able too. I have a LOT of ideas for this.. And eventually it will do more
then just the OpenSim.ini file. I hope to make it where it creates any of
the alterable files.

Time is my enemy right now. but people will come, if I build it..
