Very simple.

1. Upload all textures you wish to provide to a single avatar account.
2. Create a single working folder with optional subfolders to prepare a
logically organized collection of textures for presentation to the user.
3. save iar *only that folder* (the working folder mentioned previously)
that contains the textures you wish to make available in the library.
4. move/copy the iar file per Diva's instructions.
