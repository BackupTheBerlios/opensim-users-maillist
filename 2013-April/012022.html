Some of the best mesh avatar resources, including working rigged meshes and
tutorials can be found at the following link.

Avatar Meshes <http://blog.machinimatrix.org/avatar-workbench/>

A LOT of the information regarding mesh issues prior to blender release 2.6
is now obsolete.   The tutorials at the linked site are being revised to
reflect the new blender features.    If you have the latest version of
blender ( 2.66 ) , exporting a rigged mesh is as simple as selecting the
meshes ( Head/Torso/Legs AND the armature/skelton ), select export, and in
the OPERATOR PRESETS for the dae export, select SECOND LIFE RIGGED.

You are on your own for the skins....    I'm not sure what you mean by
greyscale reference skin...   maybe you mean skin design templates???   I
have always used these...

Robin Wood UV templates<http://www.robinwood.com/Catalog/Technical/SL-Tuts/SLPages/AVUVTemplates.html>
