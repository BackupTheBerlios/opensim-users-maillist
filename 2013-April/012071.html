what i would like to see is a Sim On A Stick type setup with

1 a default region with a pond/bay
2 a few buildings to include an asset "store"
3 a set of known working Mesh Avatars (MakeHuman can be used for a good
chunk of the workflow on this)
4 a few different vehicles (boat /truck /airplane)
5 some FX type scripted objects
6 a Theater with a Media On Prim "screen"

i think the safest way to work this across sims is to pack them into one or
more IAR files.

(oh and while im wishing it would help to have the source files for all the
objects)
