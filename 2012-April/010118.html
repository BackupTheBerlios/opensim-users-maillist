Unfortunately, this is a bug in OpenSimulator 0.7.3.1 and earlier (and hence in D2).  It will appear if you have more 
than one region on the simulator (I'm guessing a mega-region counts here) and a user teleports from one region to another.

 From the looks of it, this has been around for a long time (in OpenSimulator 0.7.2 as well) but previously the error 
was only one line rather than multiple.

I have fixed this in git master f213f55 but I'm afraid the only solution in OpenSimulator 0.7.3.1 or earlier is to 
disable the web stats module or to manually apply this patch and recompile.
