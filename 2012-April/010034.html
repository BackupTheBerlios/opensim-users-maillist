I'm running opensim on a Windows 64bit system and overall it seems to work
ok.  I recently had an issue with intermittent system crashing after
loading a large OAR file that turned out to be (most likely) an issue with
Opensim running out of memory.

The full thread/discussion is at:
http://opensim-users.2152040.n2.nabble.com/Intermittent-crashing-System-OutofMemoryException-td7281298.html


The solution was to separate out the regions into multiple opensim
instances.

This problem occurred with the 0.7.2 release, but I understand the 0.7.3
release allows you to run opensim as a 64 bit process which takes better
advantage of memory resources, so not sure which version you're running,
but if you haven't upgraded to 0.7.3 yet, might try that and see if the
issues persist.

Hope that helps!
