So the overall issue was jumping too far.  The Rutgers regions are at
11,000, and Pathlandia is around 5000. I have JumpNexus sim at about
7200. Which explained originally why it was only "teleport home."  In
any case, I was allowed to teleport to an invalid location via
landmark or script. The symptom was the world wouldn't draw in.

Landmarks wouldn't let me return home because home is too far. But I
can select "teleport home." I successfully telport back to the home
region. The home region won't draw in, and I can see the regions
surrounding it. I can teleport to another region in the same world,
then fly back to my home region, and it will redraw properly.

Kinda convoluted but it explains what happened.
