I am pretty sure using this Minimal CD installation and setup is exactly
the same as using the full CD/DVD set for installing Ubuntu, here is a
quote from the link you gave

"The Minimal CD will download packages from online archives at installation
time instead of providing them on the install CD itself."

the only difference is it downloads the files from a public repository such
as mirrors.kernel.org or something similar instead of installing them from
the DVD media, otherwise its pretty much 100% identical as installing from
the full Ubuntu DVD.
