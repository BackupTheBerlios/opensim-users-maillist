Hi  Gökhan,

Try going into the Male Avatar's inventory and creating all new body parts
(Skin, Shape, Eyes, Hair) and wear the new pieces, then you should be able
to edit the appearance, including the gender.  You can also create new
clothes and attachments, and everything you add to the Male Avatar's
appearance will be copied when your new users select the male avatar from
the Wifi web page.

Let us know if that works!

> Did you actually create an account called "Male Avatar"?  To use the Diva
>> Wifi starter avatars, you actually have to create accounts with the names
>> specified in the .ini file (and dress them as you want them to appear for
>> new users if you're so inclined).  You will need to create one account for
>> each of the starter avatars, Male Avatar, Female Avatar, and Neutral
>> Avatar. The Wifi module will then copy that account's settings for the new
>> account users create through the web front end.
>>
>> You can create the accounts on the console with the "create user"
>> command, and you're not limited to those names or just three starter avs,
>> you can have as many as you want and call them whatever you want, so long
>> as the correct information is detailed in the .ini settings.
