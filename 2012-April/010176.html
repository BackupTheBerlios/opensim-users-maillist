Maybe it's a good idea to backup the database as well as making OAR & IAR
files? I'm new at the VW aspects of this, but have been doing data storage
& DB stuff for decades. Data corruption does happen, especially with hard
disks, which can start corrupting data for some time before finally failing
completely. CD/DVD-Recordable is still a good choice for long-term
archiving for valuable data.
