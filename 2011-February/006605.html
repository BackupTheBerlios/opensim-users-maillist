Hi,

on a megaregion you can't upload the terrain files via the viewer. This will
always just change the terrain of the root region. The terrains for the
remaining child regions need to be changed via the console, by changing to
the according region with the >change region< command and loading the
terrain with >terrain load<. I discussed that in the comments on my
megaregion tutorial a bit more in-depth:
http://tgib.co.uk/2010/06/05/how-to-create-megaregion-terrain-raw-files-for-second-life-and-opensim/
