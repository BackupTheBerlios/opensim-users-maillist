http://sourceforge.net/projects/bitcoin/
The bitcoin software is distributed under the MIT License (GPL compatible)
The software could be integrated into the Opensim software, along with an
option for downloading other currency's support software.(unlike bitcoin,
those would need to be d/l from the currency company, and the users would
need to make agreements)

With  bitcoin, there is no need for the sim owner or the users to enter into
any "deal" with external money companies. That, plus an open source license,
make it a good fit as an  Opensim default currency.
http://en.wikipedia.org/wiki/Bitcoin
http://www.mtgox.com/
