I'm posting a link to a preliminary version of my OpenSim services 'care 
package'.  To recap, it contains everything needed to (manually) install 
Groups, Profiles, Search and Offline Messages using a single database 
and is assembled from existing modules that seem to work the best under 
most conditions.  It's not 100% complete as far as detailed instructions 
go (which I'm working on) and I still haven't tested it under IIS or on 
a MAC (which I'm also working on), but there's a quickstart guide and 
all the necessary files are included.  All comments, corrections and 
contributions are welcome and desired.  Hopefully someone will find this 
useful :)

The file is named 'Opensim_Services.zip' - there's also a checksum/virus 
scan log for the properly paranoid...

http://public.odosys.net/files/opensim/os_modules/
