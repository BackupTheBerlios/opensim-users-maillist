Hi Wayne,

I think I remember having this problem before and the issue was that I
hadn't completely removed the MySQL data.  If you're on Windows, just
uninstalling doesn't remove all the tables, you have to also delete the
data files to start from scratch.

See
http://dev.mysql.com/doc/refman/5.1/en/windows-installer-uninstalling.html

I believe once I got back to a clean starting point, then I was able to
re-assign the estates..

Good luck!
