Hi all,

One of the things I've found very frustrating in my quest to install
various Opensim modules is that the instructions on the wiki for each
module tend to assume that the user is on a linux system, so it will say
things like "use the command below to checkout the code" or something
similar.

This is not helpful for someone not familiar with SVN nor is it
self-explanatory for how to "get" the modules for Windows users.  I found
the instructions at

http://www.codinghorror.com/blog/2008/04/setting-up-subversion-on-windows.html
very
helpful for pulling the modules - obviously I didn't install the SVN server
part, I just followed the instructions for installing TortoiseSVN.


Hope this is helpful for other Windows users or those who haven't used SVN
before..
