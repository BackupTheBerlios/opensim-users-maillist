Hello all,

I am having an issue with creating groups on my OpenSim grid.  I am using
the most current release of the Diva Distro r16915, running my standalone in
hypergrid mode on the Windows platform.

I have used multiple browsers (Ascent, Hypo, Firefox, Cool and the SL
viewer) to attempt to create a group but to no avail.  The issue is this:

When I attempt to create a group, I keep receiving the message that it costs
100L to create the group.  Since my grid does not use money, I cannot create
the group obviously.  How do I change this?

After some research, I made these additions/changes to the MyWorld.ini:

Added under  [Startup]

*economymodule = BetaGridLikeMoneyModule* 

and added to the same file at the very bottom:

*[Economy]
    ;; The economy module is not enabled by default and there is no easy,
    ;; menu controlled way to do this. To enabled this module, insert
    ;; economymodule = BetaGridLikeMoneyModule
    ;; into the startup section (at the top of this file).
    ;; Then configure the options below as needed.

    ;; Enables selling things for $0
    SellEnabled = "false"

    ;; Money Unit fee to upload textures, animations etc
    PriceUpload = 0

    ;; Money Unit fee to create groups
    PriceGroupCreate = 0*


The following changes had no effect.  I just want to create a group! lol. 
Any ideas?

Thanks.

Raven
