If you're running the opensim server on the same Pc of the viewer you can
choose : Quick Grid Select ----> Local

Otherwise you need to insert the IP address of the server on which is
running opensim and the appropriate port.

Other details like user/password need to be the same of what you enter when
you start your server before.
