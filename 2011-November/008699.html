Hi Ravenmyst,

Group functionality doesn't work with the out-of-the-box opensim
distribution.  You have to install a separate groups module in order to
create and use groups.  (The same also applies to profiles, offline IMs,
and search.)

Fortunately, Mimetic Core just put together a nice package of all four
optional features based on existing modules, and I found it relatively
straightforward to install.

Download the module package at:
http://public.odosys.net/files/opensim/os_modules/

See the discussion about the package at:
http://opensim-users.2152040.n2.nabble.com/Re-opensim-edu-OpenSim-services-bundle-test-td7011881.html

You can see the step-by-step process I followed to install it on a
Windows/IIS system at the 11/19/11 entry:
http://fleep.wikispaces.com/FleepGrid


Hope that helps and best of luck!
