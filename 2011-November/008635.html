"Not sure if you typed that backwards in your email but did it correctly
when you were creating the database, but assuming your database name and
database username are both opensim, then it should be:

grant all on opensim.* to 'opensim'@'localhost';

Not localhost at opensim..."

That is a real easy mistake to make and Chris is correct, in the command
line the syntax IS 'opensim'@'localhost';
with no spaces around the @ and all the ' and semi colon as shown.

I prefer a GUI interface for administering the mySQL database, but have
done both command line and GUI.
Don't forget that if you are using the command line, you'll need to log in
as root in the terminal. I run openSUSE linux now so in that it's; su
Also, it's real easy to forget to START mySQL if it's stopped since it
runs in the background mostly unseen.

It's easy to verify the grant and user have been taken, in linux mysql
after logging in to it I just type; SHOW GRANTS FOR opensim at localhost;
