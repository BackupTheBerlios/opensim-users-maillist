Wayne - if you're using Mimetic Core's bundle of services, the path in
those URLs is not correct.  It should just be
http://yourdomain.com/osprofile/profile.php unless you changed something
about the file structure in the package.

Also, you only use "localhost" in the database configuration files,
everywhere else in the opensim .ini files you should be using the domain
name, for example http://fleepgrid.com/osprofile/profile.php.

Hope that helps!
