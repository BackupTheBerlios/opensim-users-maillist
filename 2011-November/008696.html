great to hear, Kurt.  all credit goes to the developers in the Flotsam 
project.  my only goal was to make installing this and other services 
easier to understand and obtaining the best components more convenient.  
as for the search/profile binaries, each OpenSim instance which will be 
connecting to the grid services needs its own copy of the binaries 
installed in the /bin folder.

these should run fine on a Linux/Mono setup.  that's my primary 
environment and the first to be tested.  if you look in the 
'readme_first' document you'll see the conditions under which these 
modules performed successfully (up to the level of functionality 
verified on the OpenSim wiki):

- Ubuntu 11.xx | OpenSUSE 11.x | CentOS 5
Mono 2.6.x/2.10.5, MySQL 5.1.x | PHP 5.3.x | Apache 2.2.x

- Windows XP SP3 (32 bit)
.NET Framework 3.5, Mysql 5.1.x | PHP 5.3.x | Apache 2.2.x (as services)

- Windows 7/Server 2008 R2 (64 bit)
.NET Framework 3.5, MySQL 5.5.x | PHP 5.3.x | Apache 2.2.x (as a WAMP stack)

- Core/Jason
