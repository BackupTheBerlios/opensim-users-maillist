I've always looked at each grid's currency to be similar to RL
currency in that the currency has different values according to which
nation you are in. As far as I know, each nation's currency is
measured against the US dollar. In each grid, the value of the grid's
currency is measured against the US dollar. In RL, you can go to a
currency exchange to change your funds to whatever currency you need.
In the metaverse, you can get the local currency through PayPal or
credit card.

I've never used BitCoin. I've checked them out in the past, before
getting into the metaverse, and I don't like the lack of information
regarding the security of my transactions. I've not had an opportunity
to check out Virwox.

In SL I use lindies, in InWorldz I use izzies and my grid will use
creds. I don't mind using one type on one grid another on a different
grid. It makes each one a bit more unique.
