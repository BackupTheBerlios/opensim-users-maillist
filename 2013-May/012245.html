You need to enable grid gods in the OpenSim.ini

in the section called :

[Permissions]

make the following true

allow_grid_gods = true

Then in the user database you need to assign your "Admin" users a
UserLevel of 255 and UserFlag of 780 this will essentially make them
have the same powers a "Linden" would in the Second Life grid.

Then you just need to type "Ctrl+Alt+G" to enable god mode in the viewer

Hope this helps, if you have any questions let us know.
