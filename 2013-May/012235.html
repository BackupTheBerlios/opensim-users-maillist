In terms of raw number of objects, you can see this as the "Active Objects" stat in any viewer, or by using the "show 
scene" command on a simulator console.

On OpenSimulator 0.7.5 (I think) there are also commands to show objects in certain areas (see "help show object pos"). 
  Unfortunately, there aren't the useful commands at the moment to show objects according to certain criteria (e.g. all 
objects with physics enabled).  These need either extra options or perhaps ideally some kind of sql like query language 
(though that's a much bigger project).

You might have better luck with the in-world solutions Kim suggests, though I can't remember using them myself.
