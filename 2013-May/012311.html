Hi Rick.  I'm very surprised to hear that since the default behaviour on OAR load is to wipe the region before the load. 
  There is a --merge option but this isn't the default.

IARs are a different matter - you will get duplicate inventory items if you don't delete the old ones first.  There 
isn't any option to do a replace here.

You can manually wipe a region with the console command

delete object name --regex .*

but this shouldnt' be necessary.
