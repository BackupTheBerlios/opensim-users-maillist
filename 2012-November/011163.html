Hi Hashim,

The best add-on module for web based account creation/management and
replacing the default avatars with your own custom avatars is Diva's Wifi
for ROBUST module.  See the wiki page for more information :
http://opensimulator.org/wiki/Wifi

In order to use this tool, you will need to run Opensim in grid mode, not a
standalone implementation.  I think the instructions Diva provides are some
of the best documentation on any 3rd party add-on module, so be sure to
read the documentation carefully, but if you run into any trouble or have
questions, just let us know.
