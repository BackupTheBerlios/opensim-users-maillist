i had this same problem earlier if i remember correctly if you are setup as
a megaregion
you can go to the ROOT region (which should be the southwest corner) and use

terrain load-tile c:\path\image.png 4 4 %rootX% %rooty% to load your map
png (it has to be a greyscale png  1024X1024)
