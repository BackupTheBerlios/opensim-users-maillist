This may be due to a bug that existed until fairly recently where inventory folder versions were not incremented 
properly.  I can imagine that it would cause these kind of symptoms if one were concurrently using two different viewer 
installations for the same account.  In theory, the problem should also be resolved after relog when it occurs.

This is fixed in master code and the fix is in the latest osgrid release.
