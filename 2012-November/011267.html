If you reread the original post, you will see that his program will
be able to download in the way Second inventory does it; the asset
server URI is for deduplicationg UPLOAD.

Download is vie LLUDP/Caps and can bypass permissions completely. In
an open source tool, grid TOS respecting code can be trivially
commented. The distribution of such a modified version to the public
would mean anyone being able to copy without deeper knowledge of
grid technologies and tools. The people who would modify the code in
that way are not people who respect copyleft licenses like the AGPL
and the people who would use the modified version will likely be
those who respect no licenses whatsoever. An open source content
ripping tool would be abused in so many ways and form the basis of a
whole new generation of super copybots.
