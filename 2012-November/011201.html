This is best investigated as a Mantis bug report.  Please open one and include the usual details (Windows/Linux, level 
of mono (if applicable), OpenSimulator version, etc.)

If you are running mono, it is much better if you run with --debug so that these traces get line numbers and the problem 
can be better identified.  AFAIK, the overhead of --debug is minimal.
