Hi,

• A stupid question : did you create the initial "opensim" MySQL  
database before running "OpenSim.exe" ?

That need to be done. The "empty" database will be populated when  
running OpenSim for the first time.

• Another point : in the logs below, you have  
"OpenSim.Framework.Data.MySQL.dll" ... Please check your OpenSim.ini  
and xx_Config.xml files ...
you need to change "OpenSim.Framework.Data.MySQL.dll" into  
"OpenSim.Data.MySQL.dll" (( It's valid only if you are using a recent  
SVN.SnapShot ))

Regards,

Ursula.
