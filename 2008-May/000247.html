Just as a heads up, if you see the error

no implementation for interface method 
OpenSim.Framework.IUserData::GetUserAppearance(LLUUID) in class 
OpenSim.Data.DB4o.DB4oUserData

when you try to start the current build, then try deleting 
Opensim.Data.Db4o.dll from bin before restarting.  This worked for me 
under mono on linux - the cause is probably the recent removal of the 
old db4o db implementation.
