> > ;; All the other settings below only really apply to the

> > ;; GroupsModule.
> >
> > ;; This module can use a PHP XmlRpc server from the Flotsam project at

> > ;; http://code.google.com/p/flotsam/

> > Module = GroupsModule

> >
> >
