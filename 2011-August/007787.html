For HG via the Map of via osTeleportAgent scripts, in the very recent 
versions of OpenSim 0.7.2 dev master you might want to note that the 
old style e.g."hg.osgrid.org:80:Vue-6400" no longer works. It now 
requires the new form e.g. "http://hg.osgrid.org:80/ Vue-6400" and 
seems to need the / on the end of the loginuri

I noticed that the old style stopped working between r/16363 
(23-Jul-2011) and r/16512 (1-Aug-2011) so you might want to check out 
any HG scripts you are using to check they still operate.
