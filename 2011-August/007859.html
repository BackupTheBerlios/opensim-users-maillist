Actually, you can run Opensim in Grid Mode and use Diva's Wifi and she does
provide basic instructions for setting it up.  For an example of it working,
can check out http://fleepgrid.com, see the Create Account page.  Note that
I did customize some of the HTML to match my branding but otherwise it's
pretty much Wifi out of the box.

To download the Wifi and instructions (in one of the readme files, I
believe), see:  https://github.com/diva/d2/downloads and make sure to grab
the package labeled "Wifi for Robust server" for the correct version of
Opensim you're running (she has zips for both 0.7.0.2 and 0.7.1).

If I recall correctly, I found the instructions fairly straightforward, just
make sure to read them carefully.

Best of luck!
