OSGrid have provided a number of files for what I assume is their 
Profile setup
 
http://download.osgrid.org/
 
osgrid_osprofile.zip for sources of the OpenProfileModule, 
.ini.example files and web server profile.php
 
OpenSimProfile.Modules.dll is the precompiled version to put in the 
bin directory
 
The following section goes in OpenSim.ini (up to 0.7.1.1 - and all 
standalones I think) but in GridCommon.ini for recent 0.7.2
 
[Profile]
    ; Change it to your own HTTP server to have the Profile server work
    Module = "OpenProfileModule"
    ProfileURL = http://profiles.osgrid.org/profile.php
