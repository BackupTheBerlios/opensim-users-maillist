On 04/08/11 13:18, Gary Banham wrote:
> yes to answer you question about groups I have been able to get the floatsam
> groups module to work
> however I won't work in the standard wamp or lamp server stack I tried for a
> couple of months
> I searched for an xmlrpc server on the web and it led me to the zend server
> I downloaded there community version and set it up on a small desktop
> I only used the  flotsam_XmlRpcGroup portion of the the project and set it
> in the server root
> I then configured the config file to point to a debase I named groups
> I also imported the sql tables into the database
>
> if your standalone has been connected to osgrid then the groups should be
> configured in it
> the only thing you need to change is the url in the opensim.ini file
>    GroupsServerURI = http://xxx.xx.xx.xx/XmlRpcGroupsServer/xmlrpc.php
> this will need to point to your groups server
>
