Make sure that enabled = false for the [WebStats] config section if you have one.  If it's not enabled you shouldn't be 
seeing these messages.

On one of the previous memory issues, you could also make sure you're using the Flotsam asset cache and that the memory 
cache part is disabled through config.  I'm not sure this was the default on 0.7.1.1.
