It just means the OpenSim bin directory for the host where your 
Robus.exe server is running.  Its likely to be same host that your 
OpenSim.exe us running on.

It just means the WifiPages directory should be at same level as bin 
(not inside it) and the provided extra bin files should go inside the 
OpenSim bin directory.

The instructions are trying to be precise in case you run Robust.exe 
and OpenSim.exe on several different hosts.
