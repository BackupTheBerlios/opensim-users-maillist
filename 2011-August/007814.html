The terminology sure can be confusing at times, especially with the 
words "Standalone" and "Grid".

The diva distribution running Wifi with Robust IS a Grid, which means it 
is a multi-user virtual world.

> I was told by someone in OSGrid that you couldn't use WiFi with a Grid
>

Sorry if I misunderstand your question, but as far as I know, you can't 
run a Robust Grid within another Robust Grid like OSGrid -- and another 
reason why the diva distro cannot run in OSGrid is because of the 
version difference. Diva is running the stable version, while OSGrid is 
running the dev branch, which is the cutting edge.
