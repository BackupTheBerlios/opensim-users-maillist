if you have more than one mysql instance (3306,3307 etc.), you can run any
kind of opensimulator with diffrent setups. 
They work separate from each other... Create database and user in each
instance and point to them in opensimulator configuration 
and make sure ports are open in router/firewall for you to be able to TP to
it..
9000-9000 for region 1 9001-9001 for region 2 etc. (or 800x, you know the
deal  )
doing it by hand is very difficult, "sandbox"  is doing all the work for
you.. 

after that START BUILDING
