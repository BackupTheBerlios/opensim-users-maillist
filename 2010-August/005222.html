Hanging at region handshake is a udp issue usually caused by the port being
blocked for that protocol.
One other potential reason i have run into into is when the machine has 2
network interfaces which allowed the udp to come in on the interface the 
simulator was not set up on.
To verify udp connectivity, not just tcp, try the port checking tool from
http://portforward.com .
