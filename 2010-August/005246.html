As announced, vComm Solutions prepared together with Snoopy and Dreamland
Metaverse the region Mumble Sandbox with mumble voice enabled.  You will
need to replace SLVoice.exe in order to use the mumble integration. The
installation instructions can be found here: http://whisper.vcomm.ch/forum.

 

The installer contains two scripts to switch between SL and mumble voice.
However, for now you will have to logout, switch and re-log again. An
automatic switch without the need to re-log is under work.

 

Even though the software seems to run very stable, the source code still
contains some wild hacks. As soon as these are cleaned up, the source code
of the client and the region module it will made available open source.
Please check for updates here:  http://whisper.vcomm.ch/forum. 

 

Cheers

Volker Gaessler
