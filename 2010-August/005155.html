I don't think that timer option for refresh is implemented, but it's trivial
to refresh using the existing timer functionality in LSL.  It won't give you
'website on a prim' unless you have access to a service to convert webpages
into images - there are several out there, or you can write your own to
install on your server.  If however you really want web-on-a-prim, MOAP
support is coming to opensim soon thanks to Justin's efforts, or you can use
the existing parcel media to show full web pages with just one tiny
change<http://metaverseheroes.helpserve.com/index.php?_m=knowledgebase&_a=viewarticle&kbarticleid=18>to
the settings.xml for the viewer.   With some cunning javascript and
some
server-side support you can also turn this parcel media into a scrollable
web browser <http://www.youtube.com/watch?v=LBfUVIhFaLc> with clickable
links. (Video was shot in SL but the same works in OpenSim if you apply the
config change to settings.xml)

Neil
