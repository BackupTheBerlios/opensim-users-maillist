with opensimulator its possible to put command line command in a (backup).txt
file and put this within opensimulator bin folder. 

something like "save oar 
C:\Users\user\Downloads\OPENSIM-Backups\my-region.oar"  
this includes saving iar files.  you don't have to remember passwords or
have typos.

when opensimulator runs, you can execute this .txt file with "command-script
backup.txt"
or put them in "startup_commands.txt" or "shutdown_commands.txt"
