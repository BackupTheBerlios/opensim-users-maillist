Ali said:

>Has any one used or know how to bring in google earth or google maps in
opensim?

Google Maps yes (as already pointed out GE is more of an application). You
can do it through parcel media or prim media, and use a script to change the
URL to the lat/long/zoom you want (you may need to make a proxy on the web
server to do this). You can see the result in SL at
http://www.daden.co.uk/pages/datascape.html and visit it at
http://www.slurl.com/secondlife/daden%20prime/160/184/26 (there should be a
TP from there - if not it's the bunker looking building in the NE corner of
the sim). The same code is working fine in OpenSim.
