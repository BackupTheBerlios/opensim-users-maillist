Well.. I don't think their viewer (yet) works on the core OpenSim base,
but this is planned:

http://www.adamfrisby.com/blog/2008/11/logging-into-an-opensim-via-realxtend-accounts/

You can use it now on top of opensim, but it defaults to the standard
LL viewer  ;(   The OGRE-based capabilities are so amazing, that when
they do link on top of opensim as a true plugin, it should immediately boost
opensim's server appeal. The undersea demo world is that incredible.

As far as getting Real's viewer to work with their version of the 
opensim server, I have
never been able to get the authentication working from behind my Linksys 
router
