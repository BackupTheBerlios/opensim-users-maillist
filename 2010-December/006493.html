The last e-mail had two ways to get a development version.

-------------------------
1. Git:

git can be installed on linux boxes with a package manager or via
http://git-scm.com/download

git can be installed on windows boxes with
http://code.google.com/p/tortoisegit/   Be aware that you might need to
download and install msysgit first.

-------------------
2. ViewGit Website interface at http://opensimulator.org/viewgit/

If you use viewgit, click 'opensim'.  Then click 'tar/gz' or 'zip' under the
actions column to the right of the version you are downloading.

>> The Mesh support is in development versions of OpenSimulator.   There

>> To get a development version, you need to use 'git' to clone the
>> repository and build from there.    If you don't want to (or don't know how
>> to use 'git'), take a look at the online 'viewgit' interface (It can take 2
>> minutes for the page to load) here:
>> http://opensimulator.org/viewgit/?a=summary&p=opensim  ... and click on
>> one of the 'zip' or the 'tar/gz' links, respectively, to the right of the
>> preferred revision listed to download a zipped or a gzipped tarball
>> version.    Remember, development versions tend to contain more bugs and
>> require you to re-do your simulator configuration.
