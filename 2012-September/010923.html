you need the landtools php files available here from OSgrid :
http://download.osgrid.org/helper_tools_opensim.zip : this needs to be in a
folder on your website configured for your grid database, i suggest making
a read only account for mysql, these do not need write access, so something
like this http://helper.osgrid.org or http://osgrid.org/helper and then
just point your viewer grid manager to this location, then you should be
able to transfer and sell land.  Let me know if you have any questions.
