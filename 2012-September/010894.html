This could be because of an issue with new avatars where OpenSimulator was not allowing multiple calling card type 
folders ("Friends" and "All") to be generated beneath the root level Calling Cards folder.

This was fixed relatively recently (in commit 7ea832d4 on Wed Aug 29 2012) but it requires an update of the ROBUST 
inventory service and it's possible that osgrid hasn't done this yet.

In the mean time, you should be able to ignore those messages unless they are affect normal folder creation.
